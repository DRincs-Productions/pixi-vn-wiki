标记语言是一种文本编码系统，用于规定文档中的结构、格式和关系。 标记语言可以控制文档的显示方式，或为自动化处理丰富其内容。 常见的标记语言包括 HTML、KML/KMZ、XML、Markdown 等。

对于 Pixi’VN，建议使用 <DynamicLink href="/start/markup-markdown">**Markdown**</DynamicLink>，因为它简单且广泛流行。 Markdown 通常可以与 HTML 结合使用，让你能够通过 CSS 实现自定义文本样式。 对于高级样式需求，可以考虑使用 CSS 框架，例如 <DynamicLink href="/start/markup-tailwindcss">**Tailwind CSS**</DynamicLink>。
