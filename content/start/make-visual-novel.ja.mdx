このチュートリアルでは、最初のビジュアルノベルを作成するプロセスを案内します。

**ビジュアルノベルとは何ですか？** ビジュアルノベル（VN）は、デジタルインタラクティブフィクションの一形態です。 ビジュアルノベルはしばしばビデオゲームのメディアと関連付けられますが、必ずしもそのようにラベル付けされるわけではありません。 They combine a textual narrative with static or animated illustrations and a varying degree of interactivity. The format is more rarely referred to as novel game, a retranscription of the wasei-eigo term noberu gēmu (ノベルゲーム), which is more often used in Japanese.

テスト目的のために、このガイドではPixi’VNを使用してビジュアルノベル[Breakdown](https://joshpowlison.itch.io/breakdown)を再現します。 Breakdownは、ビジュアルノベルが持つべきすべての特徴を備えた短編小説です。 BreakdownのクリエイターであるJosh Powlisonは、教育目的で彼のナarrationを使用することを許可してくれました❤️。

Pixi’VNは、1つ以上の<DynamicLink href="/start/narration">利用可能なナarration言語</DynamicLink>を選択することで、自分自身のナarrationを書く能力を提供します。そのため、各開発ステップで現在利用可能な言語ごとに例を作成します。

## 新しいプロジェクトを作成する

最初のステップは、新しいプロジェクトを作成することです。 You can find more information on how to create a new project starting from a template <DynamicLink href="/start#project-initialization">here</DynamicLink>. 「ビジュアルノベル - React」というテンプレートを使用します。

`Visual Novel -> React`

作成が完了したら、プロジェクトのルートにある `README.md` ファイルを読むことが非常に重要です。 このファイルには、プロジェクトに関する重要な情報とその使用方法が含まれています。

私たちの場合、プロジェクトを開始するには、次のコマンドを実行するだけで済みます。

```bash
npm install
npm start
```

## キャラクターの作成

これからこの物語のキャラクターを定義します。 これを行うために、`/values/characters.ts` ファイルで使用するキャラクターを定義します。 キャラクターの作成と使用に関する詳細情報は、<DynamicLink href="/start/character">キャラクター</DynamicLink>を参照してください。

`mc`とは何ですか？ `mc`は「メインキャラクター」の一般的な略語です。 ビジュアルノベルでは、`mc`をメインキャラクターの名前として使用するのが一般的な方法です。

```ts tab="values/characters.ts"
import { RegisteredCharacters } from "@drincs/pixi-vn";
import Character from "../models/Character";

export const mc = new Character('mc', {
    name: 'Me',
});

export const james = new Character('james', {
    name: 'James',
    color: "#0084ac"
});

export const steph = new Character('steph', {
    name: 'Steph',
    color: "#ac5900"
});

export const sly = new Character('sly', {
    name: 'Sly',
    color: "#6d00ac"
});

RegisteredCharacters.add([mc, james, steph, sly]);
```

```ts tab="App.tsx"
// Remember to import the character file at least once into your project. // [!code focus]
import "./values/characters"; // [!code focus]

export default function App() {
return // ...
}
```

### Split the narrative into labels

非常に長い`ラベル`（線形の視覚小説でも）を作成することは推奨されませんが、複数の小さな`ラベル`を作成し、必要に応じて<DynamicLink href="/start/labels-flow">ナレーションフロー制御機能</DynamicLink>を使用して「呼び出す」ことが推奨されます。

このため、私たちの場合、物語は線形であっても、2つの`labels`に分かれます。最初のものは私たちが作成したばかりのもので、2番目のものは`second_part`と呼ばれます。

これは例です：

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"
=== start ===
james: You're my roommate's replacement, huh?
james: Don't worry, you don't have much to live up to. Just don't use heroin like the last guy, and you' fine!
mc: ...

He thrusts out his hand.

james: James!
mc: ...Peter.

// ...
-> second_part

=== second_part ===

She enters my room before I'VE even had a chance to. \\n\\n...I could've just come back and gotten the platter later...
She sets it on a desk. I throw my two paper bags down beside the empty bed.

steph: They got you a new mattress, right? That last guy was a druggie, did James tell you that?
sly: *We're* the reason he got expelled!
steph: Sly! If word gets out about that... well, actually, it wouldn't matter, *he's* the one who shot himself up.

I'm fumbling for a new subject.

// ...
-> DONE
```

## キャラクター情報を編集し、それを変数として使用します。

これからプレイヤーに`mc`の名前を変更する能力を与えます。

これを行うために、プレイヤーに<DynamicLink href="/start/input">Pixi’VNの機能を使用して入力ボックスを完成させる</DynamicLink>ように依頼します。

入力値を取得した後、得られた値を使用して<DynamicLink href="/start/character#edit">キャラクター名を設定する</DynamicLink>ことができます。

これは例です：

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"
=== start ===
// ...

He thrusts out his hand.
# request input type string default Peter
What is your name?
# rename mc { _input_value_ }

// ...
-> DONE
```

```ts tab="Typescript" title="labels/startLabel.ts" groupId="narrative_language"
const startLabel = newLabel("start", [
    // ...
    () => { narration.dialogue = `He thrusts out his hand.` },
    () => {
        narration.requestInput({ type: "string" }, "Peter")
        narration.dialogue = `What is your name?`
    },
    () => {
        mc.name = narration.inputValue as string
    },
    // ...
]);
export default startLabel;
```

## 対話の「接着剤」機能を使用する

視覚小説では、現在の対話にテキストを貼り付けることがよく役立ちます。 例えば、会話を一時停止し、その後の`step`で続けることができます。 これを行うために、<DynamicLink href="/start/dialogue#dialogue-glue">接着機能</DynamicLink>を使用できます。

これは例です：

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"
=== start ===
// ...

james: Ooh, [mc]! Nice, firm handshake!
<>The last guy always gave me the dead fish.
<>I already think we're gonna get along fine.
james: Come on in and...

// ...
-> DONE
```

```ts tab="Typescript" title="labels/startLabel.ts" groupId="narrative_language"
const startLabel = newLabel("start", [
    // ...
    async () => narration.dialogue = { character: james, text: `Ooh, ${mc.name}! Nice, firm handshake!` },
    async () => {
        narration.dialogGlue = true
        narration.dialogue = `The last guy always gave me the dead fish.`
    },
    async () => {
        narration.dialogGlue = true
        narration.dialogue = `I already think we're gonna get along fine.`
    },
    async () => narration.dialogue = { character: james, text: `Come on in and...` },
    // ...
]);
export default startLabel;
```

<Callout type="info">

アセット（画像、GIF、動画など）をロードして操作するには `Assets`を使用する必要があります。 `Assets`は多くの機能を持つクラスで、PixiJSライブラリから来ています。詳細については[こちら](https://pixijs.com/8.x/guides/components/assets)をお読みください。

最初のステップの一つは、視覚小説のアセットを保存する場所を選ぶことです。 この場合、アセットはFirebaseストレージ（ホスティングサービス）に保存します。 お好きなホスティングサービスを使用することもできますし、プロジェクト内にアセットをローカルに保存することもできます。詳しくは<DynamicLink href="/start/assets">こちら</DynamicLink>をご覧ください。

## アニメーションの作成

視覚小説をよりダイナミックにするために、アニメーションを使用できます。 アニメーションの使用方法についての詳細は<DynamicLink href="/start/canvas-animations-effects">こちら</DynamicLink>を参照してください。

多くのプロパティを設定する必要がある場合は、TypeScriptの使用をお勧めします。これにより、より多くの制御、機能、および型フィードバックが得られます。

この例では、私のアニメーションが`steph`をシーンから取り出し、次の`step`で再挿入します。 彼女が正しい方向を向いていることを確認するために、x軸で彼女をミラーします。

`steph`を出したり入れたりするために、`moveOut`関数と`moveIn`関数を使用します。 ミラー効果のために、`canvas.animate`関数を使用します。

まず、`canvas.animate`関数を使用してアニメーションを作成し、`scaleX`プロパティを`-1`（ミラー）から`1`に設定します。`autoplay: false`を指定すると、すぐには開始されません。

その後、`moveIn`関数を使用して`steph`をシーンに移動させ、アニメーションの`tickerId`を渡します。これにより、トランジションが完了した後に再開できます。

次の`step`が実行される前にアニメーションが完了するように、`forceCompleteBeforeNext: true`オプションを使用します。 `moveIn`はトランジションであるため、`forceCompleteBeforeNext`はデフォルトでtrueです。

このアニメーションにはTypeScriptを使用しているので、他の言語からも呼び出せるように`label`を作成しました。

```ts title="labels/animation01.ts"
import { canvas, ImageContainer, moveIn, newLabel } from "@drincs/pixi-vn";

export const animation01 = newLabel("animation_01", [
    async () => {
        let tickerId = canvas.animate<ImageContainer>(
            "steph",
            {
                scaleX: 1,
            },
            { autoplay: false, forceCompleteBeforeNext: true }
        );

        await moveIn(
            "steph",
            {
                value: ["fm02-body", "fm02-eyes-joy", "fm02-mouth-smile01"],
                options: { xAlign: 0.8, yAlign: 1, scale: { y: 1, x: -1 }, anchor: 0.5 },
            },
            { direction: "right", ease: "easeInOut", tickerIdToResume: tickerId }
        );
    },
]);
```

<Callout type="info" title="ink">

As explained <DynamicLink href="/ink/label#use-the-call-script">here</DynamicLink>, from _ink_ you can call `labels` written in ts and vice versa.

</Callout>

これで、この`label`（`animation_01`）をメインの`label`（`start`）から呼び出すことができます。

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"

```

```ts tab="Typescript" title="labels/startLabel.ts" groupId="narrative_language"
const startLabel = newLabel("start", [
    // ...
    async () => {
        await showImageContainer("james", ["m01-body", "m01-eyes-grin", "m01-mouth-grin00"]);
        await showImageContainer("sly", ["fm01-body", "fm01-eyes-smile", "fm01-mouth-smile00"]);
        await showImageContainer("steph", ["fm02-body", "fm02-eyes-upset", "fm02-mouth-nervous00"]);
        moveOut("steph", { direction: "left", speed: 300 });

    async (props) => {
        narration.dialogGlue = true;
        narration.dialogue = ` and returns with a HUGE tinfoil-covered platter.`;
        await narration.call(animation01, props);
    },
    // ...
]);
export default startLabel;
```
