Puoi distribuire il tuo gioco su [itch.io](https://itch.io/). Si tratta di una piattaforma che ti consente di caricare il tuo gioco e distribuirlo al pubblico. È un'ottima piattaforma per distribuire il tuo gioco e ricevere riscontri dalla community.

## Gioco riproducibile sul browser

Su itch.io puoi abilitare l'opzione `Questo file sarà riprodotto nel browser` perché un file caricato sia riproducibile nel browser.

Tuttavia, è possibile eseguire soltanto un file html o js, non una directory. Ad esempio, se carichi un file zip contenente la cartella generata dopo una compilazione, vedrai un risultato simile al seguente:

![immagine](https://github.com/user-attachments/assets/0482a6fa-8c21-4fa6-b4e1-04f05bc4315d)

La soluzione è <DynamicLink href="/start/distribution-website#hosting-and-deploying">ospitare il gioco su un server</DynamicLink>, quindi creare un'iframe su itch.io per mostrarlo.

Ad esempio, dopo aver ospitato il gioco su un server, puoi creare un file index.html con i seguenti contenuti:

```html title="index.html"
<!doctype html>
<html lang="en" style="height: 100%; width: 100%;">
  <head></head>
  <body
    style="height: 100%; width: 100%; display: flex; overflow: hidden; margin: 0; background-color: #242424;"
  >
    <div id="root"
      style="height: 100%; width: 100%;"
    >
      <iframe
        src="https://pixi-vn-visual-novel-example.pages.dev/"
        style="height: 100%; width: 100%; border: none;"
        allowfullscreen
      >
      </iframe>
    </div>
  </body>
</html>
```

Quindi, puoi caricare il file index.html su itch.io.
