Un langage de balisage est un système d'encodage textuel qui définit la structure, la mise en forme et les relations au sein d'un document. Le balisage peut contrôler la façon dont un document est affiché ou enrichir son contenu pour un traitement automatisé. Parmi les langages de balisage courants, on trouve HTML, KML/KMZ, XML, Markdown, et autres.

Pour Pixi’VN, il est recommandé d'utiliser <DynamicLink href="/start/markup-markdown">**Markdown**</DynamicLink> en raison de sa simplicité et de sa popularité. Markdown peut souvent être combiné avec HTML, vous permettant d'utiliser CSS pour un style de texte personnalisé. Pour un style avancé, envisagez d'utiliser un framework CSS tel que <DynamicLink href="/start/markup-tailwindcss">**Tailwind CSS**</DynamicLink>.
