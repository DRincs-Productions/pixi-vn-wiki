---
title: Animations and transitions
---

Animating components in Pixi'VN is really simple. The core animation function is <DynamicLink href="/start/canvas-motion">`animate`</DynamicLink>. It is a wrapper around the popular [`motion`](https://motion.dev/) library, which provides a simple and powerful way to animate components. `motion` is only used to perform mathematical calculations during the animation while what triggers the change events are the <DynamicLink href="/start/canvas-tickers">Tickers</DynamicLink>.
