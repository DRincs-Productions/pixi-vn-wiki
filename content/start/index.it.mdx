Puoi iniziare a utilizzare Pixi'VN [inizializzando un nuovo progetto](#project-initialization) oppure [installando il pacchetto](#installation) in un progetto esistente.

## Prerequisiti

Prima di iniziare, è necessario avere installati i seguenti strumenti:

* [Node.js](https://nodejs.org/) versione 18 o superiore.
* Editor di testo con supporto a TypeScript, come:
  * [Visual Studio Code](https://code.visualstudio.com/)
  * [Cursor](https://www.cursor.com/)
  * [VSCodium](https://vscodium.com/)
* (Consigliato) [Git](https://git-scm.com/)
  * Un [account GitHub](https://github.com/) - Potrai utilizzare Copilot (assistente AI), la generazione automatica di pacchetti da distribuire e aggiungere commenti alla wiki

## Project initialization

Se vuoi partire da un nuovo progetto, puoi usare il seguente comando per inizializzare un nuovo progetto con i template di Pixi'VN:

```npm
npm create pixi-vn@latest
```

Puoi vedere l'elenco dei template disponibili e delle demo interattive <DynamicLink href="/start/templates">qui</DynamicLink>.

Dopo aver inizializzato il progetto, apri la directory del progetto con il tuo editor di testo (si consiglia VSCode) e inizia a sviluppare il progetto.

Tutti i templates includono un file `README.md` con maggiori informazioni sul progetto.

## Installazione

Per installare il pacchetto Pixi'VN in un progetto JavaScript esistente, utilizzare uno dei seguenti comandi:

```npm
npm install @drincs/pixi-vn
```

### Inizializzare

Prima di utilizzare l'engine Pixi'VN, è necessario inizializzare il gioco. È possibile farlo chiamando il metodo [`Game.init`](/jsdoc/pixi-vn/index/namespaces/Game/functions/init).

```ts title="src/main.tsx"
import { Game } from "@drincs/pixi-vn";

const body = document.body;
if (!body) {
    throw new Error("body element not found");
}

Game.init(body, {
    height: 1080,
    width: 1920,
    backgroundColor: "#303030",
}).then(() => {
    // ...
    Game.start("start", {});
});

// read more here: https://pixi-vn.web.app/start/other-narrative-features.html#how-manage-the-end-of-the-game
Game.onEnd(async (props) => {
    Game.clear();
    // navigate to main menu
});

Game.addOnError((error, props) => {
    console.error(`Error occurred`, error);
});

Game.onNavigate((path) => navigateTo(path));
```
