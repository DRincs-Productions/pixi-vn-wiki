## Ajouter

Pour ajouter un composant au canevas du jeu, vous pouvez utiliser [`canvas.add`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#add).

```ts tab="content/labels/start.label.ts"
import { Assets, canvas, newLabel, Sprite } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        const sprite = new Sprite();
        const texture = await Assets.load("egg_head");
        sprite.texture = texture;
        canvas.add("sprite", sprite); // [!code focus]
    },
]);
```

```ts tab="assets/index.ts"
import { AssetsManifest } from "@drincs/pixi-vn";

/**
 * Manifest for the assets used in the game.
 * You can read more about the manifest here: https://pixijs.com/8.x/guides/components/assets#loading-multiple-assets
 */
export const manifest: AssetsManifest = {
    bundles: [
        {
            name: "start",
            assets: [
                {
                    alias: "egg_head",
                    src: "https://pixijs.com/assets/eggHead.png",
                },
            ],
        },
    ],
};
```

## Obtenir

Pour obtenir un composant du canevas du jeu, vous pouvez utiliser [`canvas.find`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#find).

```ts tab="content/labels/start.label.ts"
import { Assets, canvas, newLabel, Sprite } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        const sprite = new Sprite();
        const texture = await Assets.load("egg_head");
        sprite.texture = texture;
        canvas.add("sprite", sprite);
    },
    () => {
        const sprite = canvas.find("sprite"); // [!code focus]
        if (sprite) {
            sprite.x = 100;
            sprite.y = 100;
        }
    },
]);
```

## Tout supprimer

Pour supprimer tous les composants du canevas du jeu, vous pouvez utiliser [`canvas.removeAll`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#removeAll).

```ts tab="content/labels/start.label.ts"
import { Assets, canvas, newLabel, Sprite } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        const texture = await Assets.load("egg_head");
        for (let i = 0; i < 3; i++) {
            const sprite = new Sprite();
            sprite.texture = texture;
            sprite.x = i * 150;
            canvas.add(`sprite${i}`, sprite);
        }
    },
    () => {
        canvas.removeAll(); // [!code focus]
    },
]);
```

<Accordion title="Ajouter un écouteur à un événement" id="add-a-listener-to-an-event">

:::tip

Si possible, essayez d'utiliser <DynamicLink href="/start/interface">HTML et PixiJS UI</DynamicLink> pour ajouter des boutons ou d'autres éléments avec des événements.

:::

Dans Pixi’VN, par rapport à PixiJS, vous pouvez utiliser un [écouteur avec la méthode `on`](https://pixijs.com/8.x/examples/events/click) (par exemple pour le rendre cliquable), mais si vous n'utilisez pas le décorateur [`@eventDecorator`](/jsdoc/pixi-vn/index/functions/eventDecorator), l'événement ne sera pas enregistré dans le système d'événements de Pixi’VN, donc si vous chargez une sauvegarde où cet événement a été utilisé, il sera perdu.

```ts tab="content/canvas/events.ts"
import { eventDecorator, FederatedEvent, Sprite } from "@drincs/pixi-vn";

export default class Events {
    @eventDecorator()
    static buttonEvent(event: FederatedEvent, sprite: Sprite): void {
        switch (event.type) {
            case "pointerdown":
                sprite.scale.x *= 1.25;
                sprite.scale.y *= 1.25;
                break;
        }
    }
}
```

```ts tab="content/labels/start.label.ts"
import { newLabel, showImage } from "@drincs/pixi-vn";
import Events from "../canvas/events";

export const startLabel = newLabel("start", [
    async () => {
        const bunny = await showImage("bunny", "bunny", {
            align: 0.5,
            anchor: 0.5,
        });
        // Opt-in to interactivity
        bunny.eventMode = "static";
        // Shows hand cursor
        bunny.cursor = "pointer";
        // Pointers normalize touch and mouse (good for mobile and desktop)
        bunny.on("pointerdown", Events.buttonEvent);
    },
]);
```
