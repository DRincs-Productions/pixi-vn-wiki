<Accordion title="¿Qué es Markdown?" id="what-is-markdown">

**Markdown** es un lenguaje de marcado ligero para crear texto con formato utilizando un editor de texto sin formato. John Gruber creó Markdown en 2004, en colaboración con Aaron Swartz, como un lenguaje de marcado diseñado para ser fácil de leer en su forma de código fuente. Markdown se usa ampliamente en blogs y mensajería instantánea, y también se utiliza en foros en línea, software colaborativo, páginas de documentación y archivos readme.

</Accordion>

</Accordions>

Pixi’VN no está vinculado a ningún marcado y proporciona al desarrollador la capacidad de elegir el marcado que prefiera. Sin embargo, se recomienda usar Markdown.

:::tip[Plantillas]

En todas las plantillas, el componente Markdown ya está definido.

:::

:::tip[*ink*]

Puedes usar este método con la sintaxis *ink*. Ver más <DynamicLink href="/ink/markup#markdown">aquí</DynamicLink>.

:::

A continuación se muestran algunos ejemplos de implementaciones de Markdown en el ecosistema JavaScript:

```tsx tab="React"
// I use the react-markdown library to convert the Markdown to HTML
// read more about it here: https://www.npmjs.com/package/react-markdown
import Markdown from "react-markdown";
import rehypeRaw from "rehype-raw";
import remarkGfm from "remark-gfm";

export default function MarkdownComponent({ text }: { text: string }) {
    return (
        <Markdown remarkPlugins={[remarkGfm]} rehypePlugins={[rehypeRaw]}>
            {text}
        </Markdown>
    );
}
```

```vue tab="Vue"
<!-- I use the vue-markdown-render library to convert the Markdown to HTML -->
<!-- read more about it here: https://www.npmjs.com/package/vue-markdown-render -->
<template>
    <div>
        <vue-markdown :source="src" />
    </div>
</template>

<script lang="ts">
import VueMarkdown from 'vue-markdown-render'

export default defineComponent({
  name: 'MyComponent',
  components: {
    VueMarkdown
  },
  setup(props, ctx) {
    const src = ref('---
title: header')
---

    return {
      src
    }
  }
})
</script>
```



## React Markdown Typewriter

[React Markdown Typewriter](https://www.npmjs.com/package/react-markdown-typewriter) es una biblioteca que combina Markdown y el efecto de máquina de escribir. Esta biblioteca fue creada por mí para satisfacer mi necesidad de añadir un efecto de máquina de escribir al componente Markdown en mis plantillas de React.

Si estás usando React, te recomiendo que la uses:

```tsx title="React"
import { MarkdownTypewriter } from "react-markdown-typewriter";
import rehypeRaw from "rehype-raw";
import remarkGfm from "remark-gfm";

export default function MarkdownComponent({ text }: { text: string }) {
    return (
        <MarkdownTypewriter
            remarkPlugins={[remarkGfm]}
            rehypePlugins={[rehypeRaw]}
        >
            {text}
        </MarkdownTypewriter>
    );
}
```
