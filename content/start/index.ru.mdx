Вы можете начать использовать Pixi’VN, [инициализировав новый проект](#project-initialization) или [установив пакет](#installation) в существующий проект.

## Предварительные требования

Перед началом вам необходимо установить следующие инструменты:

- [Node.js](https://nodejs.org/) версии 18 или выше.
- Текстовый редактор с поддержкой TypeScript, например:
  - [Visual Studio Code](https://code.visualstudio.com/)
  - [Cursor](https://www.cursor.com/)
  - [VSCodium](https://vscodium.com/)
- (Рекомендуется) [Git](https://git-scm.com/)
  - A [GitHub аккаунт](https://github.com/)

## Инициализация проекта

Если вы хотите начать с нового проекта, вы можете использовать следующую команду для инициализации нового проекта с шаблонами Pixi’VN:

```sh tab="npm"
npm create pixi-vn@latest
```

```sh tab="yarn"
yarn create pixi-vn
```

```sh tab="pnpm"
pnpm create pixi-vn
```

```sh tab="bun"
bun create pixi-vn
```

```sh tab="deno"
deno init --npm pixi-vn
```

Вы можете увидеть список доступных шаблонов и интерактивных демонстраций <DynamicLink href="/start/templates">здесь</DynamicLink>.

После инициализации проекта откройте каталог проекта в вашем текстовом редакторе (рекомендуется VSCode) и начните разрабатывать ваш проект.

Все шаблоны включают файл `README.md` с дополнительной информацией о проекте.

## Установка

Чтобы установить пакет Pixi’VN в существующем проекте JavaScript, используйте одну из следующих команд:

```sh tab="npm"
npm install @drincs/pixi-vn
```

```sh tab="yarn"
yarn add @drincs/pixi-vn
```

```sh tab="pnpm"
pnpm add @drincs/pixi-vn
```

```sh tab="bun"
bun add @drincs/pixi-vn
```

```sh tab="deno"
deno install npm:@drincs/pixi-vn
```

For a better developer experience, use the Vite plugin.

This plugin provides the following features:

- Endpoints to manage characters, labels, and assets manifest.
  - `GET /pixi-vn/characters` - Retrieve the list of registered characters.
  - `POST /pixi-vn/characters` - Update the list of registered characters.
  - `GET /pixi-vn/labels` - Retrieve the list of registered labels.
  - `POST /pixi-vn/labels` - Update the list of registered labels.
  - `GET /pixi-vn/assets/manifest` - Retrieve the assets manifest.
  - `POST /pixi-vn/assets/manifest` - Update the assets manifest.

### Инициализация

Перед использованием движка Pixi’VN вам необходимо инициализировать игру. Вы можете сделать это, вызвав метод `Game.init`.

This function has the following parameters:

- `element`: HTML-элемент, к которому будет добавлен холст.
- `options`: Эквивалентно параметрам, которые вы можете использовать при инициализации [PixiJS Application](https://pixijs.com/8.x/guides/basics#creating-an-application). Следующие параметры обязательны:
  - `width`: Ширина холста.
  - `height`: Высота холста.
- `devtoolsOptions`: Эквивалентно параметрам, которые вы можете использовать при инициализации <DynamicLink href="/start/canvas#use-pixijs-devtools-with-pixivn">PixiJS Devtools</DynamicLink>.

```ts title="src/main.tsx"
import { Game } from "@drincs/pixi-vn";

// Canvas setup with PIXI
const body = document.body
if (!body) {
    throw new Error('body element not found')
}

Game.init(body, {
    height: 1080,
    width: 1920,
    backgroundColor: "#303030",
}).then(() => {
    // ...
});

// read more here: https://pixi-vn.web.app/start/other-narrative-features.html#how-manage-the-end-of-the-game
Game.onEnd(async (props) => {
    Game.clear();
    props.navigate("/");
});

Game.onError((type, error, { notify }) => {
    notify("allert_error_occurred");
});
```
