<Callout title="Integration in development" type="warn">

Currently, the Live2D integration is **not yet available** in Pixi'VN. Development is in progress, and the feature may change significantly before its first stable release.

To speed up development, consider [supporting the project on Patreon](https://www.patreon.com/cw/pixi_vn). If you are interested or want to stay updated, feel free to write in the chat below!

</Callout>

**What is Live2D?** Live2D is a technology that allows 2D illustrations to move naturally, creating fluid and expressive animations without the need to redraw every frame. It is widely used in visual novels, games, and interactive applications to bring characters to life.

You can learn more about Live2D on the [official Live2D website](https://www.live2d.com/).

## Implementation plan

The Live2D integration in Pixi'VN will be built on top of [untitled-pixi-live2d-engine](https://github.com/Untitled-Story/untitled-pixi-live2d-engine) — note that this is **not an official PixiJS plugin**, but a community-maintained engine that bridges Live2D with PixiJS.
