[`VideoSprite`](/jsdoc/pixi-vn/index/classes/VideoSprite) コンポーネントは [`ImageSprite`](/jsdoc/pixi-vn/index/classes/ImageSprite) コンポーネントを継承しているため、[`ImageSprite`](/jsdoc/pixi-vn/index/classes/ImageSprite) のすべてのメソッドとプロパティを使用できます。 キャンバスに単一のビデオを表示するために使用します。

```ts tab="main.ts"
import { canvas, VideoSprite } from "@drincs/pixi-vn";

let video = new VideoSprite(
    {
        anchor: { x: 0.5, y: 0.5 },
        x: 100,
        y: 100,
    },
    "film",
);

await video.load();
canvas.add("my_video", video);
```

```json tab="src/assets/manifest.gen.json"
{
    "bundles": [
        {
            "name": "video",
            "assets": [
                {
                    "alias": "film",
                    "src": "https://pixijs.com/assets/video.mp4"
                }
            ]
        }
    ]
}
```

[`ImageSprite`](/jsdoc/pixi-vn/index/classes/ImageSprite) コンポーネントと比べて、[`VideoSprite`](/jsdoc/pixi-vn/index/classes/VideoSprite) は以下の機能を追加します：

* [`loop`](/jsdoc/pixi-vn/index/classes/VideoSprite#loop)：ビデオが終了後にループするかどうかを示します。
* [`paused`](/jsdoc/pixi-vn/index/classes/VideoSprite#paused)：ビデオが一時停止中かどうかを示します。
* [`pause`](/jsdoc/pixi-vn/index/classes/VideoSprite#pause)：ビデオを一時停止するメソッド。
* [`play`](/jsdoc/pixi-vn/index/classes/VideoSprite#play)：ビデオを再生するメソッド。
* [`currentTime`](/jsdoc/pixi-vn/index/classes/VideoSprite#currenttime)：ビデオの現在の再生時間。
* [`restart`](/jsdoc/pixi-vn/index/classes/VideoSprite#restart)：ビデオを最初から再生し直すメソッド。

## 表示

キャンバスにビデオを表示する最も簡単な方法は、[`showVideo`](/jsdoc/pixi-vn/index/functions/showVideo) 関数を使用することです。 この関数は [`load`](/jsdoc/pixi-vn/index/classes/ImageSprite#load) と [`canvas.add`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#add) を組み合わせたものです。

```ts tab="content/labels/start.label.ts"
import { newLabel, showVideo } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        // Show the videos on the canvas
        let video1 = await showVideo("video");
        // Show the video with a different alias and position
        let video2 = await showVideo("video2", "video", {
            xAlign: 0.5,
        });
    },
]);
```

## 追加

キャンバスにビデオを追加するには、[`addVideo`](/jsdoc/pixi-vn/index/functions/addVideo) 関数を使用します。 この関数はコンポーネントをキャンバスに追加するだけで、表示やテクスチャの読み込みは**行いません**。 コンポーネントをキャンバスに追加するために [`canvas.add`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#add) を使用します。

```ts tab="content/labels/start.label.ts"
import { addVideo, canvas, VideoSprite, newLabel } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    () => {
        // Add the videos to the canvas
        let video1 = addVideo("video");
        // Add the video with a different alias and position
        let video2 = addVideo("video2", "video", {
            xAlign: 0.5,
        });
    },
    async () => {
        let video1 = canvas.find<VideoSprite>("video");
        let video2 = canvas.find<VideoSprite>("video2");
        // Load the textures
        video1 && (await video1.load());
        video2 && (await video2.load());
    },
]);
```

## 削除

他のキャンバスコンポーネントと同様に、[`canvas.remove`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#remove) 関数を使用してこのコンポーネントを削除できます。

## 再生と一時停止

[`play`](/jsdoc/pixi-vn/index/classes/VideoSprite#play) および [`pause`](/jsdoc/pixi-vn/index/classes/VideoSprite#pause) メソッドを使用するか、[`paused`](/jsdoc/pixi-vn/index/classes/VideoSprite#paused) プロパティを設定します。

```ts tab="content/labels/start.label.ts"
import {
    canvas,
    narration,
    newLabel,
    showVideo,
    VideoSprite,
} from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        narration.dialogue = "add video";
        await showVideo("video");
    },
    async () => {
        narration.dialogue = "pause video";
        let video = canvas.find<VideoSprite>("video");
        if (video) {
            video.pause();
            // or: video.paused = true
        }
    },
    async () => {
        narration.dialogue = "resume video";
        let video = canvas.find<VideoSprite>("video");
        if (video) {
            video.play();
            // or: video.paused = false
        }
    },
]);
```

## ループ

[`loop`](/jsdoc/pixi-vn/index/classes/VideoSprite#loop) プロパティを設定してビデオを繰り返し再生します。

```ts tab="content/labels/start.label.ts"
import { newLabel, showVideo } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        let video = await showVideo("video");
        video.loop = true;
    },
]);
```

## 最初から再生

[`restart`](/jsdoc/pixi-vn/index/classes/VideoSprite#restart) メソッドを使用して再生を最初から開始します。

```ts tab="content/labels/start.label.ts"
import {
    canvas,
    narration,
    newLabel,
    showVideo,
    VideoSprite,
} from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        narration.dialogue = "add video";
        await showVideo("video");
    },
    async () => {
        narration.dialogue = "restart video";
        let video = canvas.find<VideoSprite>("video");
        if (video) {
            video.restart();
        }
    },
]);
```
