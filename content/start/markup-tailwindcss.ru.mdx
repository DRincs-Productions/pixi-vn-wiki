<Accordion title="Что такое Tailwind CSS?" id="what-is-tailwindcss">

Tailwind CSS — это CSS-фреймворк подхода «utility-first» для быстрого создания пользовательских дизайнов. Это низкоуровневый фреймворк, предоставляющий набор утилитарных классов, которые можно использовать для построения дизайна, не выходя из HTML. Подробнее можно узнать на [официальном сайте Tailwind CSS](https://tailwindcss.com/).

Существуют различные плагины для Tailwind CSS. Вот некоторые из них:

* [tailwind-animations](https://tailwind-animations.com/): Плагин Tailwind CSS от сообщества, предоставляющий коллекцию плавных анимационных утилит для микровзаимодействий (например, `animate-fade-in-down`, `animate-jump-in`, `animate-wiggle`).
* * [tw-animate-css](https://github.com/Wombosvideo/tw-animate-css): Плагин Tailwind CSS v4, портирующий популярную библиотеку [Animate.css](https://animate.style/) в виде утилитарных классов. Он предоставляет готовые анимации входа, выхода и привлечения внимания (например, `animate-fade-in`, `animate-bounce-in`, `animate-shake`), которые можно применять непосредственно к HTML-элементам.
* [@tailwindcss/typography](https://github.com/tailwindlabs/tailwindcss-typography): Официальный плагин Typography для Tailwind CSS предоставляет набор классов для добавления красивой типографики.

Рекомендуется использовать Tailwind CSS в проекте Pixi’VN для добавления стилей или анимаций к тексту диалогов.

```ts title="content/labels/start.label.ts"
import { narration, newLabel } from "@drincs/pixi-vn";

const startLabel = newLabel("start", [
    async () => {
        narration.dialogue = `The night was <span class="inline-block animate-pulse text-violet-400">still</span>.
        Elena pressed the crumpled letter against her chest, her <span class="inline-block animate-pulse text-pink-500">heart</span> pounding beneath her ribs.
        Outside, the <span class="inline-block text-sky-500">cold</span> wind <span class="inline-block animate-bounce text-sky-400">rustled</span> the curtains
        and carried all her <span class="inline-block text-slate-500 italic">secrets</span> into the <span class="inline-block animate-bounce text-indigo-500">dark</span>.`;
    },
]);
export default startLabel;
```
