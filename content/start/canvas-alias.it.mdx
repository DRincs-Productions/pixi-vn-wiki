A ogni componente aggiunto al canvas deve essere assegnato un `alias`. Un `alias` è un modo per fare riferimento a un componente tramite una stringa univoca.

<Accordions>

<Accordion title="Cos'è un alias in PixiJS?" id="what-is-an-alias">

L'`alias` corrisponde a `PixiJSComponent.label`, quindi non modificare il `label`, ma utilizza i <DynamicLink href="/start/canvas-functions">metodi forniti da Pixi’VN</DynamicLink> per modificare l'`alias`.

</Accordion>

</Accordions>

Se un componente viene aggiunto assegnando un `alias` già esistente, il nuovo componente sostituirà quello vecchio.

## Fattore di ereditarietà

Se un componente viene aggiunto usando un `alias` esistente, il nuovo componente, oltre a sostituire quello vecchio, erediterà le proprietà, il `zIndex` e i <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink> del componente precedente.

## Modificare

Per modificare l'`alias` di un componente del canvas, è possibile usare [`canvas.editAlias`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#editalias). Se all'alias è associato uno o più <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink>, verrà aggiornato automaticamente nel ticker.

```ts
import { canvas } from "@drincs/pixi-vn";

canvas.editAlias("sprite1", "sprite2");
```

## Altre funzionalità

<Accordions>

<Accordion title="Alias del layer di gioco" id="game-layer-alias">

In Pixi’VN, il layer di gioco è un componente speciale che rappresenta l'area di gioco principale in cui vengono renderizzati tutti gli elementi di gioco.

A questo componente è stato assegnato un `alias` speciale, [`CANVAS_APP_GAME_LAYER_ALIAS`](/jsdoc/pixi-vn/index/variables/CANVAS_APP_GAME_LAYER_ALIAS), utilizzato per fare riferimento al layer di gioco nello script.

Ciò è molto utile se si desidera eseguire alcune <DynamicLink href="/start/canvas-animations-effects">animazioni o effetti</DynamicLink> sull'intero layer. Alcune funzionalità non sono consentite su questo elemento, come l'eliminazione.

```ts
import { CANVAS_APP_GAME_LAYER_ALIAS, shakeEffect } from "@drincs/pixi-vn";

shakeEffect(CANVAS_APP_GAME_LAYER_ALIAS);
```

</Accordion>
