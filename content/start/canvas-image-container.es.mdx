El componente [`ImageContainer`](/jsdoc/pixi-vn/index/classes/ImageContainer) extiende el componente [`Container`](/jsdoc/pixi-vn/index/classes/Container), por lo que puede utilizar todos los métodos y propiedades de [`Container`](/jsdoc/pixi-vn/index/classes/Container). Permite agrupar múltiples imágenes en un único componente y manipularlas como una sola unidad.

Los hijos de [`ImageContainer`](/jsdoc/pixi-vn/index/classes/ImageContainer) son componentes [`ImageSprite`](/jsdoc/pixi-vn/index/classes/ImageSprite).

```ts tab="main.ts"
import { canvas, ImageContainer } from "@drincs/pixi-vn";

let james = new ImageContainer(
    {
        anchor: { x: 0.5, y: 0.5 },
        x: 100,
        y: 100,
    },
    ["m01-body", "m01-eyes", "m01-mouth"],
);

await james.load();
canvas.add("james", james);
```

```json tab="src/assets/manifest.gen.json"

```

En comparación con el componente [`Container`](/jsdoc/pixi-vn/index/classes/Container), [`ImageContainer`](/jsdoc/pixi-vn/index/classes/ImageContainer) añade las siguientes funcionalidades:

* [`load`](/jsdoc/pixi-vn/index/classes/ImageContainer#load): carga todas las URL de imágenes y establece las texturas resultantes en los hijos.
* Posicionamiento adicional: <DynamicLink href="/start/canvas-position">alineación y posición con porcentaje</DynamicLink>.

## Mostrar

La forma más sencilla de mostrar un grupo de imágenes en el canvas es utilizar la función [`showImageContainer`](/jsdoc/pixi-vn/index/functions/showImageContainer). This function combines [`load`](/jsdoc/pixi-vn/index/classes/ImageContainer#load) and [`canvas.add`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#add).

```ts tab="content/labels/start.label.ts"
import {
    canvas,
    ImageContainer,
    newLabel,
    showImageContainer,
} from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        let james = await showImageContainer(
            "james",
            ["m01-body", "m01-eyes", "m01-mouth"],
            {
                xAlign: 0.5,
                yAlign: 1,
            },
        );
    },
    () => {
        canvas.removeAllTickers();
        let tickerId = canvas.animate<ImageContainer>("james", {
            xAlign: 0,
            yAlign: 1,
        });
    },
]);
```

## Agregar

Para añadir un grupo de imágenes al canvas, utilice la función [`addImageCointainer`](/jsdoc/pixi-vn/index/functions/addImageCointainer). Esta función solo añade el componente al lienzo; **no** lo muestra ni carga su textura. Utiliza [`canvas.add`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#add) para agregar el componente al lienzo.

```ts tab="content/labels/start.label.ts"
import {
    addImageCointainer,
    canvas,
    ImageContainer,
    newLabel,
} from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    () => {
        let james = await addImageCointainer(
            "james",
            ["m01-body", "m01-eyes", "m01-mouth"],
            {
                xAlign: 0.5,
                yAlign: 1,
            },
        );
    },
    async () => {
        let james = canvas.find<ImageContainer>("james");
        james && (await james.load());
    },
]);
```

## Eliminar

Al igual que con otros componentes del lienzo, puedes eliminar este componente usando la función [`canvas.remove`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#remove).
