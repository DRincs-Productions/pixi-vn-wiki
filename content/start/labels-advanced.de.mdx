Dieser Abschnitt behandelt erweiterte <DynamicLink href="/start/labels">Label</DynamicLink>-Funktionen. Diese sind nicht erforderlich, um ein einfaches Spiel zu erstellen, können aber dabei helfen, komplexere und interaktivere Erlebnisse zu entwickeln.

<Accordions>

<Accordion title="`onLoadingLabel`" id="onloadinglabel">

[`onLoadingLabel`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onloadinglabel) ist eine Funktion, die in [`onStepStart`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onstepstart) ausgeführt wird, wenn der `step`-Index 0 ist, und auch wenn der Benutzer einen Spielstand lädt.

Wenn ein Spielstand geladen wird, werden alle [`onLoadingLabel`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onloadinglabel)-Funktionen der [`narration.openedLabels`](/jsdoc/pixi-vn/index/interfaces/NarrationManagerInterface#openedlabels) (das aktuelle `label` und alle `labels` im Stack) ausgeführt.

Dies ist beispielsweise nützlich, um sicherzustellen, dass alle im `label` verwendeten Bilder zwischengespeichert werden, bevor das `label` startet.

Sie können diese Funktion festlegen, indem Sie sie an die [`onLoadingLabel`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onloadinglabel)-Eigenschaft in den `label`-Optionen übergeben.

```ts title="content/labels/start.label.ts"
const startLabel = newLabel(
    "start",
    [
        async () => {
            await showImage("image1", "path/to/image1.png");
            await showImage("image2", "path/to/image2.png");
        },
    ],
    {
        onLoadingLabel: async (stepIndex, label) => { // [!code focus]
            await Assets.load("path/to/image1.png"); // [!code focus]
            await Assets.load("path/to/image2.png"); // [!code focus]
        }, // [!code focus]
    },
);
```

Sie können auch [`Game.onLoadingLabel`](/jsdoc/pixi-vn/index/namespaces/Game/functions/onLoadingLabel) verwenden, um das Laden aller `labels` global abzufangen.

```ts
import { Game } from "@drincs/pixi-vn";

Game.onLoadingLabel((_stepId, label) => { // [!code focus]
    Assets.backgroundLoadBundle(label.id); // [!code focus]
}); // [!code focus]
```

</Accordion>

<Accordion title="`onStepStart`" id="onstepstart">

[`onStepStart`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onstepstart) ist eine Funktion, die vor jedem `step` eines `label` ausgeführt wird. Legen Sie sie über die [`onStepStart`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onstepstart)-Eigenschaft in den `label`-Optionen fest.

```ts title="content/labels/start.label.ts"
const startLabel = newLabel(
    "start",
    [
        () => {
            narration.dialogue = "Step 1";
        },
        () => {
            narration.dialogue = "Step 2";
        },
    ],
    {
        onStepStart: (stepIndex, label) => { // [!code focus]
            console.log(`Step ${stepIndex} started`); // [!code focus]
        }, // [!code focus]
    },
);
```

Sie können auch [`Game.onStepStart`](/jsdoc/pixi-vn/index/namespaces/Game/functions/onStepStart) verwenden, um den Start eines beliebigen `step` in einem beliebigen `label` abzufangen.

```ts
import { Game } from "@drincs/pixi-vn";

Game.onStepStart((stepIndex, label) => { // [!code focus]
    console.log(`Step ${stepIndex} started`); // [!code focus]
}); // [!code focus]
```

</Accordion>

<Accordion title="Dynamische `step`-Liste" id="dynamic-step-list">

Beim Erstellen eines neuen `label` können Sie eine Funktion übergeben, die die `steps` für dieses `label` zurückgibt. Dadurch können Sie die `step`-Liste dynamisch basierend auf einer Bedingung ändern.

```ts title="content/labels/start.label.ts"
import { storage, narration, newLabel } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", () => {
    let condition = storage.getFlag("condition");
    if (condition) {
        return [
            () => {
                narration.dialogue = "Step 2";
            },
            () => {
                narration.dialogue = "Restart";
            },
        ];
    } else {
        return [
            () => {
                narration.dialogue = "Step 1";
            },
            async (props, { labelId }) => {
                storage.setFlag("condition", true);
                return await narration.jump(labelId, props);
            },
        ];
    }
});
```
