Il componente `ImageContainer` estende il componente <DynamicLink href="/start/canvas-components#base-components">`Container`</DynamicLink>, così puoi utilizzare tutti i metodi e le proprietà di `Container`. Viene utilizzato per aggiungere al canvas un componente composto da più immagini e manipolarlo come se fosse un'unica immagine.

I children di `ImageContainer` sono componenti <DynamicLink href="/start/canvas-images">`ImageSprite`</DynamicLink>.

Per inizializzare questo componente, è necessario passare i seguenti parametri:

- `options` (Opzionale): Corrisponde all'interfaccia `ImageContainerOptions<ImageSprite>`.
- `imageUrls` (Opzionale): L'array degli URL o dei percorsi delle immagini. If you have initialized the <DynamicLink href="/start/assets-management#initialize-the-asset-matrix-at-project-start">asset matrix</DynamicLink>, you can use the alias of the texture.

```ts
import { canvas, ImageContainer } from "@drincs/pixi-vn"

let james = new ImageContainer({
    anchor: { x: 0.5, y: 0.5 },
    x: 100,
    y: 100,
}, [
    'https://image.com/body.webp',
    'https://image.com/head.webp',
    'https://image.com/eyes.webp',
])

await james.load()
canvas.add("james", james)
```

Rispetto al componente `Container`, `ImageContainer` aggiunge le seguenti funzionalità:

- `load()`: Carica tutti gli URL delle immagini e imposta le texture risultanti nei figli.
- Posizioni aggiuntive: <DynamicLink href="/start/canvas-position">Allinea e posiziona con la percentuale</DynamicLink>.

## Visualizza

Il metodo più semplice e veloce per mostrare un gruppo di immagini sul canvas è utilizzare la funzione `showImageContainer`. Questa funzione è una combinazione delle funzioni `load` e <DynamicLink href="/start/canvas-functions#add-a-canvas-component">`canvas.add`</DynamicLink>.

Questa funzione restituirà un `ImageContainer`, che potrai utilizzare per manipolare il contenitore. This function has the following parameters:

- `alias`: L'<DynamicLink href="/start/canvas-alias">alias</DynamicLink> serve a identificare il componente.
- `imageUrls` (Opzionale): L'array degli URL o dei percorsi delle immagini. If you have initialized the <DynamicLink href="/start/assets-management#initialize-the-asset-matrix-at-project-start">asset matrix</DynamicLink>, you can use the alias of the texture. Se non si specificano gli URL, come URL verrà utilizzato l'alias.
- `options` (Opzionale): Corrisponde all'interfaccia `ImageContainerOptions<ImageSprite>`.

```ts tab="labels/startLabel.ts"
import { canvas, MoveTicker, newLabel, showImageContainer } from "@drincs/pixi-vn";

export const startLabel = newLabel("start_label", [
    async () => {
        let james = await showImageContainer("james", ["m01-body", "m01-eyes", "m01-mouth"], { // [!code focus]
            xAlign: 0.5, // [!code focus]
            yAlign: 1, // [!code focus]
        }); // [!code focus]
    },
    () => {
        canvas.removeAllTickers();
        let tickerId = canvas.addTicker("james", new MoveTicker({ destination: { x: 0, y: 1, type: "align" } }));
    },
]);
```

## Aggiungi

Per aggiungere un gruppo di immagini al canvas, puoi utilizzare la funzione `addImageCointainer`. È importante tenere presente che questa funzione aggiunge solo il componente al canvas, **non lo mostra e non carica la sua texture**. Questa funzione utilizza la funzione <DynamicLink href="/start/canvas-functions#add-a-canvas-component">`canvas.add`</DynamicLink> per aggiungere l'immagine al canvas.

Questa funzione restituirà un `ImageContainer`, che potrai utilizzare per manipolare il contenitore. This function has the following parameters:

- `alias`: L'<DynamicLink href="/start/canvas-alias">alias</DynamicLink> serve a identificare il componente.
- `imageUrls` (Opzionale): L'array degli URL o dei percorsi delle immagini. If you have initialized the <DynamicLink href="/start/assets-management#initialize-the-asset-matrix-at-project-start">asset matrix</DynamicLink>, you can use the alias of the texture. Se non si specificano gli URL, come URL verrà utilizzato l'alias.
- `options` (Opzionale): Corrisponde all'interfaccia `ImageContainerOptions<ImageSprite>`.

```ts tab="labels/startLabel.ts"
import { addImageCointainer, canvas, ImageContainer, newLabel } from "@drincs/pixi-vn";

export const startLabel = newLabel("start_label", [
    () => {
        let james = await addImageCointainer("james", ["m01-body", "m01-eyes", "m01-mouth"], { // [!code focus]
            xAlign: 0.5, // [!code focus]
            yAlign: 1, // [!code focus]
        }); // [!code focus]
    },
    async () => {
        let james = canvas.find<ImageContainer>("james");
        james && (await james.load());
    },
]);
```

## Rimuovi

Come per gli altri componenti canvas, puoi rimuovere questo componente utilizzando la funzione <DynamicLink href="/start/canvas-functions#remove-a-canvas-component">`canvas.remove`</DynamicLink>.
