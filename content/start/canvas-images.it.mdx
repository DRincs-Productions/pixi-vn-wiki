title: ImageSprite
description: Guide to using the ImageSprite component in Pixi’VN, including initialization, loading, adding, showing, and removing images on the canvas.
---

The `ImageSprite` component extends the <DynamicLink href="/start/canvas-components#base-components">`Sprite`</DynamicLink> component, so you can use all the methods and properties of `Sprite`. It is used to display a single image on the canvas.

Per inizializzare questo componente, è necessario passare i seguenti parametri:

- `options` (Opzionale): Le opzioni per il componente.
- `imageUrl` (Optional): The URL or path. Se hai inizializzato la <DynamicLink href="/start/assets-management#initialize-the-asset-matrix-at-project-start">matrice delle risorse</DynamicLink>, puoi utilizzare l'alias della texture.

```ts
import { canvas, ImageSprite } from "@drincs/pixi-vn"

let alien = new ImageSprite({
    anchor: { x: 0.5, y: 0.5 },
    x: 100,
    y: 100,
}, 'https://pixijs.com/assets/eggHead.png')

await alien.load()
canvas.add("alien", alien)
```

Compared to the `Sprite` component, `ImageSprite` adds the following features:

- `load()`: Loads the image URL and sets the resulting texture to the component.
- Additional positioning: <DynamicLink href="/start/canvas-position">align and position with percentage</DynamicLink>.

## Visualizza

The simplest way to show an image on the canvas is to use the `showImage` function. Questa funzione combina `load` e <DynamicLink href="/start/canvas-functions#add-a-canvas-component">`canvas.add`</DynamicLink>.

This function returns an `ImageSprite` that you can use to manipulate the component. Questa funzione ha i seguenti parametri:

- `alias`: L'<DynamicLink href="/start/canvas-alias">alias</DynamicLink> per identificare il componente.
- `imageUrl` (Optional): The URL or path. Se hai inizializzato la <DynamicLink href="/start/assets-management#initialize-the-asset-matrix-at-project-start">matrice delle risorse</DynamicLink>, puoi utilizzare l'alias della texture. If you don't provide the URL, then the alias is used as the URL.
- `options` (Opzionale): Le opzioni per il componente.

```ts tab="labels/startLabel.ts"
import { newLabel, showImage } from "@drincs/pixi-vn";

export const startLabel = newLabel("start_label", [
    async () => {
        let alien1 = await showImage("alien"); // [!code focus]
        let alien2 = await showImage("alien2", "alien", { // [!code focus]
            xAlign: 0.5, // [!code focus]
        }); // [!code focus]
        let alien3 = await showImage("alien3", "https://pixijs.com/assets/eggHead.png", { // [!code focus]
            xAlign: 1, // [!code focus]
        }); // [!code focus]
    },
]);
```

```ts tab="assets/manifest.ts"

 * Manifest for the assets used in the game.

        {
            name: "start",

                {
                    alias: "alien",

```

## Aggiungi

To add an image to the canvas, use the `addImage` function. Questa funzione aggiunge solo il componente al canvas; **non** lo mostra né ne carica la texture. It uses <DynamicLink href="/start/canvas-functions#add-a-canvas-component">`canvas.add`</DynamicLink> to add the component to the canvas.

This function returns an `ImageSprite` that you can use to manipulate the component. Questa funzione ha i seguenti parametri:

- `alias`: L'<DynamicLink href="/start/canvas-alias">alias</DynamicLink> per identificare il componente.
- `imageUrl` (Optional): The URL or path. Se hai inizializzato la <DynamicLink href="/start/assets-management#initialize-the-asset-matrix-at-project-start">matrice delle risorse</DynamicLink>, puoi utilizzare l'alias della texture. If you don't provide the URL, then the alias is used as the URL.
- `options` (Opzionale): Le opzioni per il componente.

```ts tab="labels/startLabel.ts"
import { addImage, canvas, ImageSprite, newLabel } from "@drincs/pixi-vn";

export const startLabel = newLabel("start_label", [
    () => {
        let alien1 = addImage("alien"); // [!code focus]
        let alien2 = addImage("alien2", "alien", { // [!code focus]
            xAlign: 0.5, // [!code focus]
        }); // [!code focus]
        let alien3 = addImage("alien3", "https://pixijs.com/assets/eggHead.png", { // [!code focus]
            xAlign: 1, // [!code focus]
        }); // [!code focus]
    },
    async () => {
        let alien1 = canvas.find<ImageSprite>("alien");
        let alien2 = canvas.find<ImageSprite>("alien2");
        let alien3 = canvas.find<ImageSprite>("alien3");
        // Load the textures
        alien1 && (await alien1.load());
        alien2 && (await alien2.load());
        alien3 && (await alien3.load());
    },
]);
```

## Rimuovi

Come con altri componenti canvas, puoi rimuovere questo componente utilizzando la funzione <DynamicLink href="/start/canvas-functions#remove-a-canvas-component">`canvas.remove`</DynamicLink>.
