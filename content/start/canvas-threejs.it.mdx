**Cos'è Three.js?** Three.js è una libreria JavaScript che rende facile la renderizzazione di grafiche 3D su un browser web. Utilizza WebGL per renderizzare le grafiche 3D nel browser. Three.js è una scelta popolare per creare grafiche 3D sul web.

Puoi scoprire di più su Three.js sul [sito web di Three.js](https://threejs.org/).

Poter interagire con elementi 3D può essere molto utile in molti casi.

La libreria [three-pixi](https://pixijs.com/8.x/guides/advanced/mixing-three-and-pixi#example-combining-3d-and-2d-elements) fornisce tale funzionalità a PixiJS.
