<Accordion title="Was ist Spine 2D?" id="what-is-spine-2d">

Spine 2D ist eine leistungsstarke 2D-Animationssoftware, die speziell für die Spieleentwicklung entwickelt wurde. Es verwendet ein Skelett-Animationssystem, das heißt, Charaktere und Objekte werden durch eine Hierarchie von **Knochen** animiert, die die Bewegung der angehängten Teile steuern.

Weitere Informationen zu Spine 2D findest du auf der [offiziellen Spine 2D-Website](https://it.esotericsoftware.com/).

</Accordion>

</Accordions>

Innerhalb deines **Pixi’VN**-Projekts kannst du die Spine 2D-Integration verwenden, um komplexe und flüssige Animationen für deine Charaktere und Objekte zu erstellen.
Diese Integration ist im Wesentlichen ein Wrapper um die offizielle [Spine 2D-Runtime für PixiJS](https://it.esotericsoftware.com/spine-pixi), mit der du alle Spine 2D-Funktionen direkt in deinem Pixi’VN-Projekt verwenden kannst.

## Installation

Um das Spine 2D-Paket in einem bestehenden JavaScript-Projekt zu installieren, verwende einen der folgenden Befehle:

```npm
npm install @drincs/pixi-vn-spine
```

:::danger

Die Bibliothek muss bei der Initialisierung des Spiels importiert werden, damit sie innerhalb von Lazy-Komponenten verwendet werden kann.

:::

```ts title="main.ts"
import "@drincs/pixi-vn-spine";

Game.init(body, {
    // ...
});
```

```ts title="content/ink/hashtag-commands.ts"
import { addBaseHashtagCommands } from "@drincs/pixi-vn-ink";

addBaseHashtagCommands({ bundleIds, assetAliasIds });

```

Die Hashtag-Befehle `show`, `edit` und `remove` können ebenso mit `spine` verwendet werden wie mit jedem anderen Canvas-Element.

## Verwendung

Du kannst die [`Spine`](/jsdoc/pixi-vn-spine/index/classes/Spine)-Komponente wie jede andere Pixi’VN-Komponente verwenden.
Du musst jedoch zuerst die Spine 2D-Assets (**skeleton** und **atlas**) laden, bevor du sie verwendest.

Zum Beispiel:

```ts tab="Typescript" title="main.ts" groupId="narrative_language"
import { Assets, canvas } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
const spine = new Spine({
    atlas: "spineboyAtlas",
    skeleton: "spineboySkeleton",
    xAlign: 0.5,
    yAlign: 1,
    animation: "idle",
});
canvas.add("boy", spine);
```

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"
=== start ===
# show spine boy skeleton spineboySkeleton atlas spineboyAtlas xAlign 0.5 yAlign 1 animation idle
-> DONE
```

```ts tab="src/assets/index.ts"
import { AssetsManifest } from "@drincs/pixi-vn";

/**
 * Manifest for the assets used in the game.
 * You can read more about the manifest here: https://pixijs.com/8.x/guides/components/assets#loading-multiple-assets
 */
export const manifest: AssetsManifest = {
    bundles: [
        {
            name: "start",
            assets: [
                {
                    alias: "spineboySkeleton",
                    src: "https://raw.githubusercontent.com/EsotericSoftware/spine-runtimes/4.3/examples/spineboy/export/spineboy-pro.skel",
                },
                {
                    alias: "spineboyAtlas",
                    src: "https://raw.githubusercontent.com/EsotericSoftware/spine-runtimes/4.3/examples/spineboy/export/spineboy-pma.atlas",
                },
            ],
        },
    ],
};
```

<Accordion title="Skin" id="skin">

Ein Spine-Modell kann mehrere Skins haben, und du kannst zur Laufzeit mit der Funktion [`setSkin`](/jsdoc/pixi-vn-spine/index/classes/Spine#setskin) zwischen ihnen wechseln.

```ts tab="Typescript" title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["goblinsSkeleton", "goblinsAtlas"]);
        const spine = new Spine({
            atlas: "goblinsAtlas",
            skeleton: "goblinsSkeleton",
            skin: "goblin",
            xAlign: 0.5,
            yAlign: 1,
            animation: "walk",
        });
        canvas.add("goblin", spine);
    },
    () => {
        canvas.find<Spine>("goblin")?.setSkin("goblingirl");
    },
]);
```

<Accordion title="Animation" id="animation">

Es gibt zwei Funktionen, mit denen du eine Spine-Animation abspielen kannst: [`setAnimation`](/jsdoc/pixi-vn-spine/index/classes/Spine#setanimation) und [`addAnimation`](/jsdoc/pixi-vn-spine/index/classes/Spine#addanimation).

* [`setAnimation`](/jsdoc/pixi-vn-spine/index/classes/Spine#setanimation) ersetzt sofort die aktuell auf dem Track abgespielte Animation durch die neue.
* [`addAnimation`](/jsdoc/pixi-vn-spine/index/classes/Spine#addanimation) reiht die neue Animation in eine Warteschlange ein, sodass sie erst startet, nachdem die aktuelle Animation auf dem Track beendet ist, ohne sie zu unterbrechen.

```ts tab="Typescript" title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
        const spine = new Spine({
            atlas: "spineboyAtlas",
            skeleton: "spineboySkeleton",
            xAlign: 0.5,
            yAlign: 1,
            animation: "idle",
        });
        canvas.add("boy", spine);
    },
    () => {
        canvas.find<Spine>("boy")?.addAnimation("walk", { loop: true });
    },
]);
```

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"
=== start ===
# show spine boy skeleton spineboySkeleton atlas spineboyAtlas xAlign 0.5 yAlign 1 animation idle
# pause
# play walk on spine boy loop true

-> DONE
```

<Accordion title="Spine + motion.js-Animationen" id="motion">

Du kannst Spine-Animationen mit den klassischen bewegungsbasierten Animationen von Pixi’VN ([`canvas.animate`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#animate)) kombinieren, wie im folgenden Beispiel gezeigt.

Für fortgeschrittenere Spiele kannst du Spine-Animationen auch direkt von einem Ticker steuern, der durch Ereignisse wie einen Tastendruck ausgelöst wird, anstatt dich nur auf vordefinierte Sequenzen zu verlassen.

```ts title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
        const spine = new Spine({
            atlas: "spineboyAtlas",
            skeleton: "spineboySkeleton",
            xAlign: 0,
            yAlign: 1,
            animation: "walk",
        });
        canvas.add("boy", spine);
        canvas.animate(
            spine,
            [
                [{ xAlign: 1 }, { duration: 1, ease: "linear" }],
                [{ scaleX: -1 }, { duration: 0.2 }],
                [{ xAlign: 0 }, { duration: 1, ease: "linear" }],
                [{ scaleX: 1 }, { duration: 0.2 }],
            ],
            { repeat: Infinity },
        );
    },
]);
```

<Accordion title="Animationssequenz" id="animation-sequence">

Es ist auch möglich, Animationssequenzen mit der Funktion [`playSequence`](/jsdoc/pixi-vn-spine/index/classes/Spine#playsequence) zu erstellen.

```ts title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
        const spine = new Spine({
            atlas: "spineboyAtlas",
            skeleton: "spineboySkeleton",
            xAlign: 0.5,
            yAlign: 1,
        });
        spine.playSequence([["idle", { loop: true, duration: 0.5 }], "jump"], {
            repeat: Infinity,
        });
        canvas.add("boy", spine);
    },
]);
```

<Accordion title="Tracks löschen" id="clear-tracks">

Du kannst Spine-Animationen mit zwei Methoden entfernen: [`clearTrack`](/jsdoc/pixi-vn-spine/index/classes/Spine#cleartrack), das die Animation auf einem einzelnen Track löscht, und [`clearTracks`](/jsdoc/pixi-vn-spine/index/classes/Spine#cleartracks), das die Animationen auf allen Tracks löscht.

```ts tab="Typescript" title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
        const spine = new Spine({
            atlas: "spineboyAtlas",
            skeleton: "spineboySkeleton",
            xAlign: 0,
            yAlign: 1,
            animation: "walk",
        });
        canvas.add("boy", spine);
    },
    () => {
        canvas.find<Spine>("boy")?.clearTracks();
    },
]);
```
