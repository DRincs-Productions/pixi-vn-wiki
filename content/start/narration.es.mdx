La narración es el núcleo de una novela visual. Consiste en <DynamicLink href="/start/dialogue">diálogos</DynamicLink>, <DynamicLink href="/start/choices">opciones</DynamicLink> y <DynamicLink href="/start/input">solicitudes de entrada</DynamicLink> que se muestran al jugador a medida que avanza el juego.

Con Pixi’VN, puedes usar diferentes lenguajes narrativos e incluso cambiar entre ellos para aprovechar las características de cada uno.

* **JavaScript/TypeScript**: Un lenguaje de programación que te permite escribir tu narración con acceso completo a todas las funciones del proyecto, pero puede requerir más código.
* **<DynamicLink href="/ink">*ink*</DynamicLink>**: Un lenguaje narrativo diseñado para escribir historias fácilmente, con código mínimo.
* **<DynamicLink href="/renpy">Ren'Py</DynamicLink>**: Próximamente.
