import { TextCanvas, TextCanvasStyle } from "@/components/examples";

Die [`Text`](/jsdoc/pixi-vn/index/classes/Text)-Komponente erweitert die [`PixiJS.Text`](https://pixijs.com/8.x/guides/components/scene-objects/text)-Komponente, sodass Sie alle Methoden und Eigenschaften von [`PixiJS.Text`](https://pixijs.com/8.x/guides/components/scene-objects/text) verwenden können. Sie dient dazu, Text auf dem Canvas anzuzeigen.

```ts title="main.ts"
import { canvas, Text } from "@drincs/pixi-vn";

const basicText = new Text({ text: "Basic text in pixi", align: 0.5 });

canvas.add("text", basicText);
```

Im Vergleich zur [`PixiJS.Text`](https://pixijs.com/8.x/guides/components/scene-objects/text)-Komponente fügt [`Text`](/jsdoc/pixi-vn/index/classes/Text) die folgenden Funktionen hinzu:

* Zusätzliche Positionierung: <DynamicLink href="/start/canvas-position">Ausrichten und Positionieren mit Prozentangaben</DynamicLink>.

## Anzeigen

Der einfachste Weg, Text auf dem Canvas anzuzeigen, ist die Verwendung der Funktion [`showText`](/jsdoc/pixi-vn/index/functions/showText).

```ts tab="content/labels/start.label.ts"
import { newLabel, showText } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        let text = await showText("text", "Hello World!", {
            xAlign: 0.5,
            yAlign: 0.5,
        });
        text.style.fontSize = 30;
    },
]);
```

## Entfernen

Wie bei anderen Leinwand-Komponenten kann diese Komponente mit der Funktion [`canvas.remove`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#remove) entfernt werden.

## Stil

Um den Text zu gestalten, verwenden Sie [`TextStyle`](/jsdoc/pixi-vn/index/variables/TextStyle). Diese Klasse ermöglicht es Ihnen, Schriftfamilie, Größe, Farbe, Kontur, Schatten und vieles mehr anzupassen.

```ts tab="content/labels/start.label.ts"
import { canvas, newLabel, Text, TextStyle } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    () => {
        const skewStyle = new TextStyle({
            fontFamily: "Arial",
            dropShadow: {
                alpha: 0.8,
                angle: 2.1,
                blur: 4,
                color: "0x111111",
                distance: 10,
            },
            fill: "#ffffff",
            stroke: { color: "#004620", width: 12, join: "round" },
            fontSize: 60,
            fontWeight: "lighter",
        });

        const skewText = new Text({
            text: "SKEW IS COOL",
            style: skewStyle,
            align: 0.5,
            skew: { x: 0.65, y: -0.3 },
        });

        canvas.add("text", skewText);
    },
]);
```
