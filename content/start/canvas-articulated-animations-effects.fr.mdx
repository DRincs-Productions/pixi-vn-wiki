Les animations articulées sont des fonctions qui utilisent la fonction [`canvas.animate`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#animate) pour créer des animations pouvant être appliquées aux composants canvas. Ces fonctions sont généralement utilisées pour créer des effets comme le tremblement, le rebond, ou d'autres animations complexes impliquant plusieurs étapes ou composants.

<Accordions>

<Accordion title="Tremblement" id="shake">

La fonction [`shakeEffect`](/jsdoc/pixi-vn/index/functions/shakeEffect) est une animation articulée qui fait trembler un composant.

```ts tab="content/labels/start.label.ts"
import {
    CANVAS_APP_GAME_LAYER_ALIAS,
    newLabel,
    shakeEffect,
    showImage,
} from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        await showImage("bg", "bg", { scale: 1.3 });
        await showImage("alien", "alien", { align: 0.5 });
        shakeEffect("alien"); // [!code focus]
    },
    async () => {
        shakeEffect(CANVAS_APP_GAME_LAYER_ALIAS); // [!code focus]
    },
]);
```
