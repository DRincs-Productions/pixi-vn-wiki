A cada componente añadido al canvas se le debe asignar un `alias`. Un `alias` es una forma de hacer referencia a un componente mediante una cadena única.

<Accordions>

<Accordion title="What does `alias` correspond to in PixiJS?" id="what-is-an-alias">

El `alias` corresponde a `PixiJSComponent.label`, por lo que no debes cambiar el `label`, sino usar los <DynamicLink href="/start/canvas-functions">métodos proporcionados por Pixi’VN</DynamicLink> para cambiar el `alias`.

</Accordion>

</Accordions>

Si se añade un componente asignando un `alias` ya existente, el nuevo componente reemplazará al antiguo.

## Factor de herencia

Si se añade un componente usando un `alias` existente, el nuevo componente, además de reemplazar al antiguo, heredará las propiedades, el `zIndex` y los <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink> del componente anterior.

## Editar

Para editar el `alias` de un componente del canvas, puedes usar [`canvas.editAlias`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#editalias). Si el alias tiene uno o más <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink> asociados, se actualizará automáticamente en el ticker.

```ts
import { canvas } from "@drincs/pixi-vn";

canvas.editAlias("sprite1", "sprite2");
```

## Otras funcionalidades

<Accordions>

<Accordion title="Alias del layer de juego" id="game-layer-alias">

En Pixi’VN, el layer de juego es un componente especial que representa el área principal del juego donde se renderizan todos los elementos del juego.

A este componente se le ha asignado un `alias` especial, [`CANVAS_APP_GAME_LAYER_ALIAS`](/jsdoc/pixi-vn/index/variables/CANVAS_APP_GAME_LAYER_ALIAS), que se utiliza para hacer referencia al layer de juego en el script.

Esto es muy útil si deseas ejecutar algunas <DynamicLink href="/start/canvas-animations-effects">animaciones o efectos</DynamicLink> en todo el layer. Algunas funcionalidades no están permitidas en este elemento, como la eliminación.

```ts
import { CANVAS_APP_GAME_LAYER_ALIAS, shakeEffect } from "@drincs/pixi-vn";

shakeEffect(CANVAS_APP_GAME_LAYER_ALIAS);
```

</Accordion>
