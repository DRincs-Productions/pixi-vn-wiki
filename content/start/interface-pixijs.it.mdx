Come spiegato più dettagliatamente <DynamicLink href="/start/canvas">qui</DynamicLink>, l'intera parte di rendering di Pixi'VN è basata su PixiJS. Sfruttando i vantaggi dei <DynamicLink href="/start/interface#adding-pixijs-ui-layers">PixiJS Laters</DynamicLink>, puoi usare PixiJS per creare schermate dell'interfaccia utente.

<Callout title="Templates" type="info">
In all templates a PixiJS Layer called "ui" will be created on startup. Puoi usare questo livello per creare le schermate dell'interfaccia utente.
</Callout>

You can also combine HTML Later components with PixiJS Laters ones. To do this, you just need to add the PixiJS components during the first rendering of an HTML screen and clean up PixiJS Later when the screen is closed.

Ad esempio:

```tsx title="React"
import { useEffect } from "react";
import { Assets, canvas } from "@drincs/pixi-vn";
import { Sprite } from "pixi.js";

export default function MyScreen() {
    useEffect(() => {
        let layer = canvas.getLayer("ui");
        if (layer) {

}
```
