Каждому компоненту, добавляемому на холст, необходимо назначить псевдоним (`alias`). Псевдоним (`alias`) — это способ обращения к компоненту по уникальной строке.

<Accordions>

<Accordion title="Чему соответствует `alias` в PixiJS?" id="what-is-an-alias">

Псевдоним (`alias`) соответствует `PixiJSComponent.label`, поэтому не изменяйте `label` напрямую, а используйте <DynamicLink href="/start/canvas-functions">методы, предоставляемые Pixi’VN</DynamicLink>, для изменения псевдонима (`alias`).

</Accordion>

</Accordions>

Если компонент добавляется с назначением уже существующего псевдонима (`alias`), новый компонент заменит старый.

## Фактор наследования

Если компонент добавляется с использованием существующего псевдонима (`alias`), новый компонент, помимо замены старого, унаследует свойства, `zIndex` и <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink> старого компонента.

## Редактировать

Чтобы отредактировать псевдоним (`alias`) компонента холста, можно использовать [`canvas.editAlias`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#editalias). Если с псевдонимом (`alias`) связан один или несколько <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink>, они будут автоматически обновлены.

```ts
import { canvas } from "@drincs/pixi-vn";

canvas.editAlias("sprite1", "sprite2");
```

## Другие возможности

<Accordions>

<Accordion title="Псевдоним игрового слоя" id="game-layer-alias">

В Pixi’VN игровой слой — это специальный компонент, представляющий основную игровую область, в которой отображаются все игровые элементы.

Этому компоненту назначен специальный псевдоним (`alias`) — [`CANVAS_APP_GAME_LAYER_ALIAS`](/jsdoc/pixi-vn/index/variables/CANVAS_APP_GAME_LAYER_ALIAS), который используется для обращения к игровому слою в скрипте.

Это очень удобно, если вы хотите применить <DynamicLink href="/start/canvas-animations-effects">анимации или эффекты</DynamicLink> ко всему слою. Некоторые операции для этого элемента недоступны, например удаление.

```ts
import { CANVAS_APP_GAME_LAYER_ALIAS, shakeEffect } from "@drincs/pixi-vn";

shakeEffect(CANVAS_APP_GAME_LAYER_ALIAS);
```
