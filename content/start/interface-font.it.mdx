Questa pagina spiega come modificare il font e le dimensioni del testo dell'UI in un progetto di Pixi'VN.

<Callout title="Modelli" type="info">

I modelli di Pixi'VN utilizzano Vite e il CSS di Tailwind, quindi le seguenti istruzioni sono pensate appositamente per tali configurazioni.

</Callout>

## Importa il font

Per importare un font puoi utilizzare un servizio come [Google Fonts](https://fonts.google.com/) o ospitare localmente dei font. Per Google Fonts, aggiungi un `@import` al tuo `index.css`.

Esempio:

```css title="index.css"
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap');
```

Per i font locali, copia i file in `public/fonts/` e dichiara un `@font-face` in `index.css`.

Esempio:

```css title="index.css"
@font-face {
  font-family: 'MyCustomFont';
  src: url('/fonts/MyCustomFont-Regular.woff2') format('woff2'),
       url('/fonts/MyCustomFont-Regular.woff') format('woff');
  font-weight: 400;
  font-style: normal;
}
```

## Imposta il font e le dimensioni globali

In `index.css` (o nel tuo foglio di stile globale) imposta la famiglia del font e, se desiderato, le variabili CSS per le dimensioni.

Esempio:

```css title="index.css"
@import "tailwindcss";
@plugin "@tailwindcss/typography";
@plugin "tailwindcss-motion";

:root {
  --ui-font-family: 'Inter', ui-sans-serif, system-ui, -apple-system, 'Segoe UI', Roboto; /* [!code ++] */
  --ui-font-size: 16px; /* [!code ++] */
}

html,
body {
  background-color: #242424;
  height: 100%;
}

body {
  margin: 0;
  min-height: 100vh;
  display: flex;
  overflow: hidden;
}

@layer base { /* [!code ++] */
  html { font-size: 16px; } /* [!code ++] */
  body { font-family: var(--ui-font-family); } /* [!code ++] */
  .ui-text { font-size: var(--ui-font-size); } /* [!code ++] */
} /* [!code ++] */
```

## Utilizza le utilità di Tailwind nei componenti

Preferisci le classi di Tailwind per coerenza.

Esempio:

* React: `<div className="font-sans text-ui-base">Testo dell'UI</div>`
* Vue: `<div class="font-sans text-ui-base">Testo dell'UI</div>`

Per le parti dell'UI che necessitano di un aspetto personalizzato (finestre di dialogo, menu di scelta), aggiungi classi come `font-display text-ui-lg`.
