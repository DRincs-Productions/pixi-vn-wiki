---
title: Linguaggio di markup e CSS
description: Panoramica dei linguaggi di markup in Pixi'VN e consigli per l'utilizzo dei framework Markdown e CSS.
---

Un linguaggio di markup è un sistema di codifica del testo che specifica la struttura, la formattazione e le relazioni all'interno di un documento. Il markup può controllare il modo in cui un documento viene visualizzato o arricchirne il contenuto per l'elaborazione automatica. I linguaggi di markup più comuni includono HTML, KML/KMZ, XML, Markdown e altri.

Per Pixi'VN, si consiglia di utilizzare <DynamicLink href="/start/markup-markdown">**Markdown**</DynamicLink> per la sua semplicità e popolarità. Spesso il markdown può essere combinato con l'HTML, consentendo di utilizzare il CSS per personalizzare lo stile del testo. Per uno stile avanzato, prendi in considerazione l'utilizzo di un framework CSS come <DynamicLink href="/start/markup-tailwindcss">**Tailwind CSS**</DynamicLink>.
