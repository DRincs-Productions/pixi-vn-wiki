Pixi’VNは視覚小説をレンダリングするために[PixiJS](https://pixijs.com/8.x/guides/basics/what-pixijs-is)を使用します。 Pixi’VN APIを使用して、PixiJSキャンバス（あるいは、より良い選択として2D WebGLレンダリングエンジン）に画像、テキスト、アニメーションを追加できます。

JavaScript/TypeScriptからキャンバスと対話できるだけでなく、他のナarrative言語も使用できます：

- <DynamicLink href="/ink/ink-canvas">***ink***</DynamicLink>
- Ren'Py (Under development)

![pixijs-h2](/pixijs.svg)

## PixiJSとは何ですか？

PixiJSは、ウェブ用に利用可能な最も高速で軽量な2Dライブラリであり、すべてのデバイスで動作し、WebGLおよびWebGPUを使用して豊かでインタラクティブなグラフィックスやクロスプラットフォームアプリケーションを作成することを可能にします。 それは高速で柔軟性があり、使いやすいです。 それは[Good Pizza, Great Pizza](https://www.goodpizzagreatpizza.com/)や[The Enchanted Cave 2](https://store.steampowered.com/app/368610/The_Enchanted_Cave_2/)などのゲームで使用されています。

You can learn more about PixiJS on the [PixiJS website](https://www.pixijs.com/).

## Pixi’VNとPixiJSの違い

Pixi’VN provides a API object called `canvas` used to interface with Pixi’VN's PixiJS application. So you don't need to install pixi.js and the features of Pixi’VN is not equivalent to that of PixiJS. Although not recommended, you can use `canvas.app` to directly access the PixiJS application used by Pixi’VN.

The use of Pixi’VN, in rendering, is very similar to that of PixiJS. The only differences are as follows:

- All the components added in the canvas will be linked to an <DynamicLink href="/start/canvas-alias">alias</DynamicLink> of your choice. This alias will be used to identify the component and to manipulate it.
- Compared to PixiJS, Pixi’VN saves the current canvas state at each <DynamicLink href="/start/labels">step</DynamicLink>.
  **Attention**: when saving the canvas state, only components linked to an alias will be saved (if you added components by directly accessing `PixiJS.Application`, they will not be considered).
- Pixi’VN provides <DynamicLink href="/start/canvas-functions">various functionalities</DynamicLink> to add, remove, find... components in the canvas.
- Pixi’VN provides <DynamicLink href="/start/canvas-components">their components</DynamicLink> of which some correspond to PixiJS while others are added to add some features.
- Just like for the components, the <DynamicLink href="/start/canvas-tickers-functions">tickers</DynamicLink> are managed by Pixi’VN. If you use a PixiJS ticker, the ticker state will not be saved.
- You can't set a listener with the `on` method, but must use the <DynamicLink href="/start/canvas-functions#add-a-listener-for-a-given-event">onEvent</DynamicLink> to add a listener.

## Use PixiJS DevTools with Pixi’VN

[**PixiJS DevTools**](https://pixijs.io/devtools/) is a [Chrome extension](https://chromewebstore.google.com/detail/pixijs-devtools/dlkffcaaoccbofklocbjcmppahjjboce) that allows you to inspect and debug PixiJS applications. You can use it to view the display list, inspect textures, and debug your PixiJS application. You can use it with Pixi’VN to inspect the canvas and debug your visual novel.

![devtools](/devtool-properties.gif)

For access to the PixiJS DevTools after installing it, you need to open the Chrome DevTools (F12) and go to the `PixiJS` tab.
