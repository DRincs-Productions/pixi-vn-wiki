Jeder Komponente, die dem Canvas hinzugefügt wird, muss ein `alias` zugewiesen werden. Ein `alias` ist eine Möglichkeit, eine Komponente über eine eindeutige Zeichenkette zu referenzieren.

<Accordions>

<Accordion title="Was ist ein Alias in PixiJS?" id="what-is-an-alias">

Der `alias` entspricht `PixiJSComponent.label`. Ändern Sie daher nicht das `label`, sondern verwenden Sie die <DynamicLink href="/start/canvas-functions">von Pixi’VN bereitgestellten Methoden</DynamicLink>, um den `alias` zu ändern.

</Accordion>

</Accordions>

Wenn eine Komponente durch Zuweisung eines bereits vorhandenen `alias` hinzugefügt wird, ersetzt die neue Komponente die alte.

## Vererbungsfaktor

Wenn eine Komponente unter Verwendung eines bestehenden `alias` hinzugefügt wird, erbt die neue Komponente zusätzlich zum Ersetzen der alten die Eigenschaften, den `zIndex` und die <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink> der alten Komponente.

## Bearbeiten

Um den `alias` einer Canvas-Komponente zu bearbeiten, können Sie [`canvas.editAlias`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#editalias) verwenden. Wenn dem Alias ein oder mehrere <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink> zugeordnet sind, wird er im Ticker automatisch aktualisiert.

```ts
import { canvas } from "@drincs/pixi-vn";

canvas.editAlias("sprite1", "sprite2");
```

## Weitere Funktionen

<Accordions>

<Accordion title="Alias des Spiel-Layers" id="game-layer-alias">

In Pixi’VN ist der Spiel-Layer eine spezielle Komponente, die den Hauptspielbereich darstellt, in dem alle Spielelemente gerendert werden.

Dieser Komponente wurde ein spezieller `alias` zugewiesen, [`CANVAS_APP_GAME_LAYER_ALIAS`](/jsdoc/pixi-vn/index/variables/CANVAS_APP_GAME_LAYER_ALIAS), der verwendet wird, um den Spiel-Layer im Skript zu referenzieren.

Das ist sehr nützlich, wenn Sie bestimmte <DynamicLink href="/start/canvas-animations-effects">Animationen oder Effekte</DynamicLink> auf den gesamten Layer anwenden möchten. Einige Funktionen sind für dieses Element nicht erlaubt, wie z. B. das Löschen.

```ts
import { CANVAS_APP_GAME_LAYER_ALIAS, shakeEffect } from "@drincs/pixi-vn";

shakeEffect(CANVAS_APP_GAME_LAYER_ALIAS);
```

</Accordion>
