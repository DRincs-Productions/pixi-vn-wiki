Un lenguaje de marcado es un sistema de codificación de texto que especifica la estructura, el formato y las relaciones dentro de un documento. El marcado puede controlar cómo se muestra un documento o enriquecer su contenido para el procesamiento automatizado. Los lenguajes de marcado comunes incluyen HTML, KML/KMZ, XML, Markdown y otros.

Para Pixi’VN, se recomienda usar <DynamicLink href="/start/markup-markdown">**Markdown**</DynamicLink> por su simplicidad y popularidad. Markdown a menudo puede combinarse con HTML, lo que te permite usar CSS para estilos de texto personalizados. Para un estilo avanzado, considera usar un framework CSS como <DynamicLink href="/start/markup-tailwindcss">**Tailwind CSS**</DynamicLink>.
