<Accordion title="Qu'est-ce que Tailwind CSS ?" id="what-is-tailwindcss">

Tailwind CSS est un framework CSS utilitaire permettant de créer rapidement des designs personnalisés. C'est un framework de bas niveau qui fournit un ensemble de classes utilitaires permettant de construire des designs personnalisés sans quitter le HTML. Vous pouvez en apprendre davantage sur le [site officiel de Tailwind CSS](https://tailwindcss.com/).

Plusieurs plugins Tailwind CSS sont disponibles. En voici quelques-uns :

* [tailwind-animations](https://tailwind-animations.com/) : Un plugin Tailwind CSS communautaire qui fournit une collection d'utilitaires d'animation fluides axés sur les micro-interactions (ex. `animate-fade-in-down`, `animate-jump-in`, `animate-wiggle`).
* [tw-animate-css](https://github.com/Wombosvideo/tw-animate-css) : Un plugin Tailwind CSS v4 qui porte la populaire bibliothèque [Animate.css](https://animate.style/) sous forme de classes utilitaires. Il expose des animations d'entrée, de sortie et d'attention prêtes à l'emploi (ex. `animate-fade-in`, `animate-bounce-in`, `animate-shake`) applicables directement aux éléments HTML.
* [@tailwindcss/typography](https://github.com/tailwindlabs/tailwindcss-typography) : Le plugin Typography officiel de Tailwind CSS fournit un ensemble de classes pour ajouter une belle typographie.

Il est recommandé d'utiliser Tailwind CSS dans votre projet Pixi’VN pour ajouter du style ou des animations à votre texte de dialogue.

```ts title="content/labels/start.label.ts"
import { narration, newLabel } from "@drincs/pixi-vn";

const startLabel = newLabel("start", [
    async () => {
        narration.dialogue = `The night was <span class="inline-block animate-pulse text-violet-400">still</span>.
        Elena pressed the crumpled letter against her chest, her <span class="inline-block animate-pulse text-pink-500">heart</span> pounding beneath her ribs.
        Outside, the <span class="inline-block text-sky-500">cold</span> wind <span class="inline-block animate-bounce text-sky-400">rustled</span> the curtains
        and carried all her <span class="inline-block text-slate-500 italic">secrets</span> into the <span class="inline-block animate-bounce text-indigo-500">dark</span>.`;
    },
]);
export default startLabel;
```
