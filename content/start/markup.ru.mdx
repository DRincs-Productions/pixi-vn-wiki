Язык разметки — это система текстового кодирования, которая определяет структуру, форматирование и связи внутри документа. Разметка может управлять отображением документа или обогащать его содержимое для автоматизированной обработки. Распространённые языки разметки включают HTML, KML/KMZ, XML, Markdown и другие.

Для Pixi’VN рекомендуется использовать <DynamicLink href="/start/markup-markdown">**Markdown**</DynamicLink> благодаря его простоте и популярности. Markdown зачастую можно комбинировать с HTML, что позволяет использовать CSS для пользовательского оформления текста. Для расширенной стилизации рассмотрите использование CSS-фреймворка, например <DynamicLink href="/start/markup-tailwindcss">**Tailwind CSS**</DynamicLink>.
