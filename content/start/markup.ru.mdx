---
title: Язык разметки
---

Язык разметки - это система кодирования текста, определяющая структуру и форматирование документа, а также потенциально возможные связи между его частями. Разметка может управлять отображением документа или обогащать его содержимое для облегчения автоматизированной обработки. Существуют различные типы языков разметки, такие как HTML, KML/KMZ, XML, Markdown и другие.

Due to its simplicity and popularity it is recommended to use <DynamicLink href="/start/markup-markdown.md">**Markdown**</DynamicLink>. Кроме того, поскольку часто возможно комбинировать Markdown с HTML, можно использовать CSS для стилизации текста. It is recommended to use a CSS framework like <DynamicLink href="/start/markup-tailwindcss">**Tailwind CSS**</DynamicLink>.
