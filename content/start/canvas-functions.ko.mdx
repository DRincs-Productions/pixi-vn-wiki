## 추가

게임 캔버스에 컴포넌트를 추가하려면 [`canvas.add`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#add)를 사용할 수 있습니다.

```ts tab="content/labels/start.label.ts"
import { Assets, canvas, newLabel, Sprite } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        const sprite = new Sprite();
        const texture = await Assets.load("egg_head");
        sprite.texture = texture;
        canvas.add("sprite", sprite); // [!code focus]
    },
]);
```

```ts tab="assets/index.ts"
        {

                {
                    alias: "egg_head",

                },
            ],
        },
    ],
};
```

## 가져오기

게임 캔버스에서 컴포넌트를 가져오려면 [`canvas.find`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#find)를 사용할 수 있습니다.

```ts tab="content/labels/start.label.ts"
import { Assets, canvas, newLabel, Sprite } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        const sprite = new Sprite();
        const texture = await Assets.load("egg_head");
        sprite.texture = texture;
        canvas.add("sprite", sprite);
    },
    () => {
        const sprite = canvas.find("sprite"); // [!code focus]
        if (sprite) {
            sprite.x = 100;
            sprite.y = 100;
        }
    },
]);
```

## 모두 제거

게임 캔버스에서 모든 컴포넌트를 제거하려면 [`canvas.removeAll`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#removeAll)를 사용할 수 있습니다.

```ts tab="content/labels/start.label.ts"
import { Assets, canvas, newLabel, Sprite } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        const texture = await Assets.load("egg_head");
        for (let i = 0; i < 3; i++) {
            const sprite = new Sprite();
            sprite.texture = texture;
            sprite.x = i * 150;
            canvas.add(`sprite${i}`, sprite);
        }
    },
    () => {
        canvas.removeAll(); // [!code focus]
    },
]);
```

<Accordion title="이벤트에 리스너 추가하기" id="add-a-listener-to-an-event">

:::tip

가능하다면 <DynamicLink href="/start/interface">HTML과 PixiJS UI</DynamicLink>를 사용하여 이벤트가 있는 버튼이나 다른 요소를 추가해 보세요.

:::

Pixi’VN에서는 PixiJS와 비교하여 [`on` 메서드를 사용하는 리스너](https://pixijs.com/8.x/examples/events/click)를 사용할 수 있지만(예를 들어 클릭 가능하게 만들기 위해), [`@eventDecorator`](/jsdoc/pixi-vn/index/functions/eventDecorator) 데코레이터를 사용하지 않으면 해당 이벤트는 Pixi’VN 이벤트 시스템에 등록되지 않으므로, 해당 이벤트가 사용된 세이브를 불러오면 그 이벤트는 사라집니다.

```ts tab="content/canvas/events.ts"
import { eventDecorator, FederatedEvent, Sprite } from "@drincs/pixi-vn";

export default class Events {
    @eventDecorator()
    static buttonEvent(event: FederatedEvent, sprite: Sprite): void {
        switch (event.type) {
            case "pointerdown":
                sprite.scale.x *= 1.25;
                sprite.scale.y *= 1.25;
                break;
        }
    }
}
```

```ts tab="content/labels/start.label.ts"
import { newLabel, showImage } from "@drincs/pixi-vn";
import Events from "../canvas/events";

export const startLabel = newLabel("start", [
    async () => {
        const bunny = await showImage("bunny", "bunny", {
            align: 0.5,
            anchor: 0.5,
        });
        // Opt-in to interactivity
        bunny.eventMode = "static";
        // Shows hand cursor
        bunny.cursor = "pointer";
        // Pointers normalize touch and mouse (good for mobile and desktop)
        bunny.on("pointerdown", Events.buttonEvent);
    },
]);
```
