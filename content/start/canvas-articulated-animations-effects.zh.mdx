关节动画是使用 [`canvas.animate`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#animate) 函数创建可应用于画布组件的动画的函数。 这些函数通常用于创建摇晃、弹跳等效果，或涉及多个步骤或组件的其他复杂动画。

<Accordion title="摇晃" id="shake">

[`shakeEffect`](/jsdoc/pixi-vn/index/functions/shakeEffect) 函数是一个使组件摇晃的关节动画。

```ts tab="content/labels/start.label.ts"
import {
    CANVAS_APP_GAME_LAYER_ALIAS,
    newLabel,
    shakeEffect,
    showImage,
} from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        await showImage("bg", "bg", { scale: 1.3 });
        await showImage("alien", "alien", { align: 0.5 });
        shakeEffect("alien"); // [!code focus]
    },
    async () => {
        shakeEffect(CANVAS_APP_GAME_LAYER_ALIAS); // [!code focus]
    },
]);
```

```ts tab="assets/index.ts"
import { AssetsManifest } from "@drincs/pixi-vn";

        },
    ],
};
```
