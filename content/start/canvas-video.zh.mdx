[`VideoSprite`](/jsdoc/pixi-vn/index/classes/VideoSprite) 组件继承自 [`ImageSprite`](/jsdoc/pixi-vn/index/classes/ImageSprite) 组件，因此您可以使用 [`ImageSprite`](/jsdoc/pixi-vn/index/classes/ImageSprite) 的所有方法和属性。 它用于在画布上显示单个视频。

```ts tab="main.ts"
import { canvas, VideoSprite } from "@drincs/pixi-vn";

let video = new VideoSprite(
    {
        anchor: { x: 0.5, y: 0.5 },
        x: 100,
        y: 100,
    },
    "film",
);

await video.load();
canvas.add("my_video", video);
```

```json tab="src/assets/manifest.gen.json"
{
    "bundles": [
        {
            "name": "video",
            "assets": [
                {
                    "alias": "film",
                    "src": "https://pixijs.com/assets/video.mp4"
                }
            ]
        }
    ]
}
```

与 [`ImageSprite`](/jsdoc/pixi-vn/index/classes/ImageSprite) 组件相比，[`VideoSprite`](/jsdoc/pixi-vn/index/classes/VideoSprite) 新增了以下功能：

* [`loop`](/jsdoc/pixi-vn/index/classes/VideoSprite#loop)：指示视频是否在播放结束后循环。
* [`paused`](/jsdoc/pixi-vn/index/classes/VideoSprite#paused)：指示视频是否已暂停。
* [`pause`](/jsdoc/pixi-vn/index/classes/VideoSprite#pause)：暂停视频的方法。
* [`play`](/jsdoc/pixi-vn/index/classes/VideoSprite#play)：播放视频的方法。
* [`currentTime`](/jsdoc/pixi-vn/index/classes/VideoSprite#currenttime)：视频的当前播放时间。
* [`restart`](/jsdoc/pixi-vn/index/classes/VideoSprite#restart)：从头重新播放视频的方法。

## 显示

在画布上显示视频最简单的方式是使用 [`showVideo`](/jsdoc/pixi-vn/index/functions/showVideo) 函数。 该函数将 [`load`](/jsdoc/pixi-vn/index/classes/ImageSprite#load) 和 [`canvas.add`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#add) 合并为一步。

```ts tab="content/labels/start.label.ts"
import { newLabel, showVideo } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        // Show the videos on the canvas
        let video1 = await showVideo("video");
        // Show the video with a different alias and position
        let video2 = await showVideo("video2", "video", {
            xAlign: 0.5,
        });
    },
]);
```

## 添加

要将视频添加到画布，请使用 [`addVideo`](/jsdoc/pixi-vn/index/functions/addVideo) 函数。 该函数仅将组件添加到画布，**不会**显示它或加载其纹理。 它通过 [`canvas.add`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#add) 将组件添加到画布。

```ts tab="content/labels/start.label.ts"
import { addVideo, canvas, VideoSprite, newLabel } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    () => {
        // Add the videos to the canvas
        let video1 = addVideo("video");
        // Add the video with a different alias and position
        let video2 = addVideo("video2", "video", {
            xAlign: 0.5,
        });
    },
    async () => {
        let video1 = canvas.find<VideoSprite>("video");
        let video2 = canvas.find<VideoSprite>("video2");
        // Load the textures
        video1 && (await video1.load());
        video2 && (await video2.load());
    },
]);
```

## 移除

与其他画布组件一样，你可以使用 [`canvas.remove`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#remove) 函数移除该组件。

## 播放与暂停

使用 [`play`](/jsdoc/pixi-vn/index/classes/VideoSprite#play) 和 [`pause`](/jsdoc/pixi-vn/index/classes/VideoSprite#pause) 方法，或设置 [`paused`](/jsdoc/pixi-vn/index/classes/VideoSprite#paused) 属性。

```ts tab="content/labels/start.label.ts"
import {
    canvas,
    narration,
    newLabel,
    showVideo,
    VideoSprite,
} from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        narration.dialogue = "add video";
        await showVideo("video");
    },
    async () => {
        narration.dialogue = "pause video";
        let video = canvas.find<VideoSprite>("video");
        if (video) {
            video.pause();
            // or: video.paused = true
        }
    },
    async () => {
        narration.dialogue = "resume video";
        let video = canvas.find<VideoSprite>("video");
        if (video) {
            video.play();
            // or: video.paused = false
        }
    },
]);
```

## 循环

设置 [`loop`](/jsdoc/pixi-vn/index/classes/VideoSprite#loop) 属性以使视频重复播放。

```ts tab="content/labels/start.label.ts"
import { newLabel, showVideo } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        let video = await showVideo("video");
        video.loop = true;
    },
]);
```

## 重新开始

使用 [`restart`](/jsdoc/pixi-vn/index/classes/VideoSprite#restart) 方法从头重新播放。

```ts tab="content/labels/start.label.ts"
import {
    canvas,
    narration,
    newLabel,
    showVideo,
    VideoSprite,
} from "@drincs/pixi-vn";

export const startLabel = newLabel("start", [
    async () => {
        narration.dialogue = "add video";
        await showVideo("video");
    },
    async () => {
        narration.dialogue = "restart video";
        let video = canvas.find<VideoSprite>("video");
        if (video) {
            video.restart();
        }
    },
]);
```
