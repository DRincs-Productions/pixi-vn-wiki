---

Narration ist der Kern eines Visual Novels. Sie besteht aus <DynamicLink href="/start/dialogue">Dialogen</DynamicLink>, <DynamicLink href="/start/choices">Auswahlmöglichkeiten</DynamicLink> und <DynamicLink href="/start/input">Eingabeaufforderungen</DynamicLink>, die dem Spieler im Spielverlauf angezeigt werden.

Mit Pixi’VN können Sie verschiedene Erzählsprachen verwenden und sogar zwischen ihnen wechseln, um die Vorteile jeder einzelnen zu nutzen.

* **JavaScript/TypeScript**: Eine Programmiersprache, mit der Sie Ihre Narration mit vollständigem Zugriff auf alle Projektfunktionen schreiben können, aber möglicherweise mehr Code erfordert.
* **<DynamicLink href="/ink">*ink*</DynamicLink>**: Eine Erzählsprache, die zum einfachen Schreiben von Geschichten mit minimalem Code entwickelt wurde.
* **<DynamicLink href="/renpy">Ren'Py</DynamicLink>**: Demnächst verfügbar.
