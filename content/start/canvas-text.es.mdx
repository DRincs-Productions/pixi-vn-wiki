---
title: Text
---

```ts
import { canvas, Text } from "@drincs/pixi-vn";

const basicText = new Text({ text: "Basic text in pixi", align: 0.5 });

canvas.add("text", basicText);
```

## Add

As with other canvas components, you can add this component using the <DynamicLink href="/start/canvas-functions#add-a-canvas-component">`canvas.add`</DynamicLink> function.

## Remove

As with other canvas components, you can remove this component using the <DynamicLink href="/start/canvas-functions#remove-a-canvas-component">`canvas.remove`</DynamicLink> function.

## Style

```ts
import { canvas, Text, TextStyle } from "@drincs/pixi-vn";

const skewStyle = new TextStyle({
    fontFamily: "Arial",
    dropShadow: {
        alpha: 0.8,
        angle: 2.1,
        blur: 4,
        color: "0x111111",
        distance: 10,
    },
    fill: "#ffffff",
    stroke: { color: "#004620", width: 12, join: "round" },
    fontSize: 60,
    fontWeight: "lighter",
});

const skewText = new Text({
    text: "SKEW IS COOL",
    style: skewStyle,
    align: 0.5,
    skew: { x: 0.65, y: -0.3 },
});

canvas.add("text", skewText);
```
