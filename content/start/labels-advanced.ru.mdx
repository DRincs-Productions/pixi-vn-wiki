Этот раздел охватывает расширенные функции <DynamicLink href="/start/labels">нарративных узлов</DynamicLink>. Они не обязательны для создания базовой игры, но могут помочь вам создавать более сложные и интерактивные впечатления.

<Accordion title="`onLoadingLabel`" id="onloadinglabel">

[`onLoadingLabel`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onloadinglabel) — это функция, выполняемая в [`onStepStart`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onstepstart), если индекс нарративного шага (`step`) равен 0, а также при загрузке пользователем файла сохранения.

При загрузке файла сохранения выполняются все функции [`onLoadingLabel`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onloadinglabel) из [`narration.openedLabels`](/jsdoc/pixi-vn/index/interfaces/NarrationManagerInterface#openedlabels) (текущий нарративный узел (`label`) и все нарративные узлы (`labels`) в стеке).

Это полезно, например, для того чтобы убедиться, что все изображения, используемые в нарративном узле (`label`), закешированы до его запуска.

Вы можете задать эту функцию, передав её в свойство [`onLoadingLabel`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onloadinglabel) в параметрах нарративного узла (`label`).

```ts title="content/labels/start.label.ts"
const startLabel = newLabel(
    "start",
    [
        async () => {
            await showImage("image1", "path/to/image1.png");
            await showImage("image2", "path/to/image2.png");
        },
    ],
    {
        onLoadingLabel: async (stepIndex, label) => { // [!code focus]
            await Assets.load("path/to/image1.png"); // [!code focus]
            await Assets.load("path/to/image2.png"); // [!code focus]
        }, // [!code focus]
    },
);
```

Вы также можете использовать [`Game.onLoadingLabel`](/jsdoc/pixi-vn/index/namespaces/Game/functions/onLoadingLabel) для глобального перехвата загрузки всех нарративных узлов (`labels`).

```ts
import { Game } from "@drincs/pixi-vn";

Game.onLoadingLabel((_stepId, label) => { // [!code focus]
    Assets.backgroundLoadBundle(label.id); // [!code focus]
}); // [!code focus]
```

<Accordion title="`onStepStart`" id="onstepstart">

[`onStepStart`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onstepstart) — это функция, выполняемая перед каждым нарративным шагом (`step`) нарративного узла (`label`). Задайте её через свойство [`onStepStart`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onstepstart) в параметрах нарративного узла (`label`).

```ts title="content/labels/start.label.ts"
const startLabel = newLabel(
    "start",
    [
        () => {
            narration.dialogue = "Step 1";
        },
        () => {
            narration.dialogue = "Step 2";
        },
    ],
    {
        onStepStart: (stepIndex, label) => { // [!code focus]
            console.log(`Step ${stepIndex} started`); // [!code focus]
        }, // [!code focus]
    },
);
```

Вы также можете использовать [`Game.onStepStart`](/jsdoc/pixi-vn/index/namespaces/Game/functions/onStepStart) для перехвата начала любого нарративного шага (`step`) в любом нарративном узле (`label`).

```ts
import { Game } from "@drincs/pixi-vn";

Game.onStepStart((stepIndex, label) => { // [!code focus]
    console.log(`Step ${stepIndex} started`); // [!code focus]
}); // [!code focus]
```

</Accordion>

<Accordion title="`onStepEnd`" id="onstepend">

[`onStepEnd`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onstepend) — это функция, выполняемая после каждого нарративного шага (`step`) нарративного узла (`label`). Задайте её через свойство [`onStepEnd`](/jsdoc/pixi-vn/index/interfaces/LabelProps#onstepend) в параметрах нарративного узла (`label`).

```ts title="content/labels/start.label.ts"
const startLabel = newLabel(
    "start",
    [
        () => {
            narration.dialogue = "Step 1";
        },
        () => {
            narration.dialogue = "Step 2";
        },
    ],
    {
        onStepEnd: (stepIndex, label) => { // [!code focus]
            console.log(`Step ${stepIndex} ended`); // [!code focus]
        }, // [!code focus]
    },
);
```

Вы также можете использовать [`Game.onStepEnd`](/jsdoc/pixi-vn/index/namespaces/Game/functions/onStepEnd) для перехвата окончания любого нарративного шага (`step`) в любом нарративном узле (`label`).

```ts
import { Game } from "@drincs/pixi-vn";

Game.onStepEnd((stepIndex, label) => { // [!code focus]
    console.log(`Step ${stepIndex} ended`); // [!code focus]
}); // [!code focus]
```

<Accordion title="Динамический список `step`" id="dynamic-step-list">

При создании нового нарративного узла (`label`) вы можете передать функцию, которая возвращает нарративные шаги (`steps`) для этого нарративного узла (`label`). Это позволяет динамически изменять список нарративных шагов (`step`) в зависимости от условия.

```ts title="content/labels/start.label.ts"
import { storage, narration, newLabel } from "@drincs/pixi-vn";

export const startLabel = newLabel("start", () => {
    let condition = storage.getFlag("condition");
    if (condition) {
        return [
            () => {
                narration.dialogue = "Step 2";
            },
            () => {
                narration.dialogue = "Restart";
            },
        ];
    } else {
        return [
            () => {
                narration.dialogue = "Step 1";
            },
            async (props, { labelId }) => {
                storage.setFlag("condition", true);
                return await narration.jump(labelId, props);
            },
        ];
    }
});
```
