Повествование — это основа визуальной новеллы. Оно состоит из <DynamicLink href="/start/dialogue">диалогов</DynamicLink>, <DynamicLink href="/start/choices">выборов</DynamicLink> и <DynamicLink href="/start/input">запросов ввода</DynamicLink>, отображаемых игроку по мере прохождения игры.

С Pixi’VN вы можете использовать различные нарративные языки и даже переключаться между ними, чтобы использовать преимущества каждого из них.

* **JavaScript/TypeScript**: Язык программирования, позволяющий писать повествование с полным доступом ко всем функциям проекта, но может потребовать больше кода.
* **<DynamicLink href="/ink">*ink*</DynamicLink>**: Нарративный язык, разработанный для лёгкого написания историй с минимальным количеством кода.
* **<DynamicLink href="/renpy">Ren'Py</DynamicLink>**: Скоро.
