Eine Auszeichnungssprache ist ein textbasiertes Kodierungssystem, das Struktur, Formatierung und Beziehungen innerhalb eines Dokuments festlegt. Auszeichnungssprachen können steuern, wie ein Dokument dargestellt wird, oder seinen Inhalt für die automatisierte Verarbeitung anreichern. Bekannte Auszeichnungssprachen sind HTML, KML/KMZ, XML, Markdown und andere.

Für Pixi’VN wird die Verwendung von <DynamicLink href="/start/markup-markdown">**Markdown**</DynamicLink> empfohlen, da es einfach und weit verbreitet ist. Markdown lässt sich häufig mit HTML kombinieren, sodass Sie CSS für individuelle Textgestaltung nutzen können. Für fortgeschrittenes Styling empfiehlt sich der Einsatz eines CSS-Frameworks wie <DynamicLink href="/start/markup-tailwindcss">**Tailwind CSS**</DynamicLink>.
