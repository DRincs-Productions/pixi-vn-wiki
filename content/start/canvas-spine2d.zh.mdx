<Accordion title="什么是 Spine 2D？" id="what-is-spine-2d">

Spine 2D 是一款专为游戏开发设计的强大 2D 动画软件。 它使用骨骼动画系统，这意味着角色和物体是通过控制附加部件运动的**骨骼**层级来进行动画的。

你可以在[Spine 2D 官方网站](https://it.esotericsoftware.com/)了解更多关于 Spine 2D 的信息。

</Accordion>

</Accordions>

在你的 **Pixi’VN** 项目中，你可以使用 Spine 2D 集成为角色和物体创建复杂而流畅的动画。
该集成本质上是对官方 [PixiJS 版 Spine 2D 运行时](https://it.esotericsoftware.com/spine-pixi) 的封装，让你可以直接在 Pixi’VN 项目中使用所有 Spine 2D 功能。

## 安装

要在现有的 JavaScript 项目中安装 Spine 2D 包，请使用以下命令之一：

```npm
npm install @drincs/pixi-vn-spine
```

:::danger

该库必须在游戏初始化时导入，以便可以在 Lazy 组件内部使用。

:::

```ts title="main.ts"
import "@drincs/pixi-vn-spine";

Game.init(body, {
    // ...
});
```

<Accordion title="ink" id="ink">

```ts title="content/ink/hashtag-commands.ts"
import { addBaseHashtagCommands } from "@drincs/pixi-vn-ink";

addBaseHashtagCommands({ bundleIds, assetAliasIds });

```

`show`、`edit` 和 `remove` 井号命令也可以与 `spine` 一起使用，就像其他任何画布元素一样。

## 使用方法

你可以像使用其他任何 Pixi’VN 组件一样使用 [`Spine`](/jsdoc/pixi-vn-spine/index/classes/Spine) 组件。
不过，在使用之前，你必须先加载 Spine 2D 资源（**skeleton** 和 **atlas**）。

例如：

```ts tab="Typescript" title="main.ts" groupId="narrative_language"
import { Assets, canvas } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
const spine = new Spine({
    atlas: "spineboyAtlas",
    skeleton: "spineboySkeleton",
    xAlign: 0.5,
    yAlign: 1,
    animation: "idle",
});
canvas.add("boy", spine);
```

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"
=== start ===
# show spine boy skeleton spineboySkeleton atlas spineboyAtlas xAlign 0.5 yAlign 1 animation idle
-> DONE
```

```ts tab="src/assets/index.ts"
import { AssetsManifest } from "@drincs/pixi-vn";

/**
 * Manifest for the assets used in the game.
 * You can read more about the manifest here: https://pixijs.com/8.x/guides/components/assets#loading-multiple-assets
 */
export const manifest: AssetsManifest = {
    bundles: [
        {
            name: "start",
            assets: [
                {
                    alias: "spineboySkeleton",
                    src: "https://raw.githubusercontent.com/EsotericSoftware/spine-runtimes/4.3/examples/spineboy/export/spineboy-pro.skel",
                },
                {
                    alias: "spineboyAtlas",
                    src: "https://raw.githubusercontent.com/EsotericSoftware/spine-runtimes/4.3/examples/spineboy/export/spineboy-pma.atlas",
                },
            ],
        },
    ],
};
```

<Accordion title="皮肤" id="skin">

一个 Spine 模型可以拥有多个皮肤，你可以使用 [`setSkin`](/jsdoc/pixi-vn-spine/index/classes/Spine#setskin) 函数在运行时切换它们。

```ts tab="Typescript" title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["goblinsSkeleton", "goblinsAtlas"]);
        const spine = new Spine({
            atlas: "goblinsAtlas",
            skeleton: "goblinsSkeleton",
            skin: "goblin",
            xAlign: 0.5,
            yAlign: 1,
            animation: "walk",
        });
        canvas.add("goblin", spine);
    },
    () => {
        canvas.find<Spine>("goblin")?.setSkin("goblingirl");
    },
]);
```

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"
# show spine goblin skeleton goblinsSkeleton atlas goblinsAtlas skin goblin xAlign 0.5 yAlign 1 animation walk
# pause

# pause
-> DONE
```

<Accordion title="动画" id="animation">

有两个函数可用于播放 Spine 动画：[`setAnimation`](/jsdoc/pixi-vn-spine/index/classes/Spine#setanimation) 和 [`addAnimation`](/jsdoc/pixi-vn-spine/index/classes/Spine#addanimation)。

* [`setAnimation`](/jsdoc/pixi-vn-spine/index/classes/Spine#setanimation) 会立即用新动画替换轨道上当前正在播放的任何动画。
* [`addAnimation`](/jsdoc/pixi-vn-spine/index/classes/Spine#addanimation) 会将新动画加入队列，使其仅在轨道上当前动画结束后才开始播放，而不会中断当前动画。

```ts tab="Typescript" title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
        const spine = new Spine({
            atlas: "spineboyAtlas",
            skeleton: "spineboySkeleton",
            xAlign: 0.5,
            yAlign: 1,
            animation: "idle",
        });
        canvas.add("boy", spine);
    },
    () => {
        canvas.find<Spine>("boy")?.addAnimation("walk", { loop: true });
    },
]);
```

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"
=== start ===
# show spine boy skeleton spineboySkeleton atlas spineboyAtlas xAlign 0.5 yAlign 1 animation idle
# pause
# play walk on spine boy loop true
# pause
-> DONE
```

<Accordion title="Spine + motion.js 动画" id="motion">

你可以将 Spine 动画与 Pixi’VN 经典的基于运动的动画（[`canvas.animate`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#animate)）结合使用，如下例所示。

对于更高级的游戏，你也可以直接通过由事件（例如按钮点击）触发的计时器来驱动 Spine 动画，而不仅仅依赖预定义的序列。

```ts title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
        const spine = new Spine({
            atlas: "spineboyAtlas",
            skeleton: "spineboySkeleton",
            xAlign: 0,
            yAlign: 1,
            animation: "walk",
        });
        canvas.add("boy", spine);
        canvas.animate(
            spine,
            [
                [{ xAlign: 1 }, { duration: 1, ease: "linear" }],
                [{ scaleX: -1 }, { duration: 0.2 }],
                [{ xAlign: 0 }, { duration: 1, ease: "linear" }],
                [{ scaleX: 1 }, { duration: 0.2 }],
            ],
            { repeat: Infinity },
        );
    },
]);
```

<Accordion title="动画序列" id="animation-sequence">

你也可以使用 [`playSequence`](/jsdoc/pixi-vn-spine/index/classes/Spine#playsequence) 函数来创建动画序列。

```ts title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
        const spine = new Spine({
            atlas: "spineboyAtlas",
            skeleton: "spineboySkeleton",
            xAlign: 0.5,
            yAlign: 1,
        });
        spine.playSequence([["idle", { loop: true, duration: 0.5 }], "jump"], {
            repeat: Infinity,
        });
        canvas.add("boy", spine);
    },
]);
```

<Accordion title="清除轨道" id="clear-tracks">

你可以使用两种方法移除 Spine 动画：[`clearTrack`](/jsdoc/pixi-vn-spine/index/classes/Spine#cleartrack)，用于清除单条轨道上的动画；以及 [`clearTracks`](/jsdoc/pixi-vn-spine/index/classes/Spine#cleartracks)，用于清除所有轨道上的动画。

```ts tab="Typescript" title="content/labels/start.label.ts" groupId="narrative_language"
import { Assets, canvas, newLabel } from "@drincs/pixi-vn";
import { Spine } from "@drincs/pixi-vn-spine";

export const startLabel = newLabel("start", [
    async () => {
        await Assets.load(["spineboySkeleton", "spineboyAtlas"]);
        const spine = new Spine({
            atlas: "spineboyAtlas",
            skeleton: "spineboySkeleton",
            xAlign: 0,
            yAlign: 1,
            animation: "walk",
        });
        canvas.add("boy", spine);
    },
    () => {
        canvas.find<Spine>("boy")?.clearTracks();
    },
]);
```

```ink tab="ink" title="ink/start.ink" groupId="narrative_language"
=== start ===
# show spine boy skeleton spineboySkeleton atlas spineboyAtlas x 0 y 1 animation walk
# pause
# clear tracks on spine boy
# pause
-> DONE
```
