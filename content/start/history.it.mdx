Pixi’VN salva tutti i dialoghi, le scelte, le risposte e altro ancora, ad ogni `step` eseguito durante il gioco.

## Ottieni

La cronologia della narrazione è un elenco di tutti i dialoghi e delle scelte mostrati al giocatore.\
Per ottenere la cronologia narrativa, utilizza `stepHistory.narrativeHistory`. Restituisce un elenco di `NarrativeHistory<T>[]`.

```typescript
const dialogues: NarrativeHistory[] = stepHistory.narrativeHistory;
```

## Rimuovi

Per eliminare tutta la cronologia narrativa, utilizza `stepHistory.removeNarrativeHistory()`.

```typescript
stepHistory.removeNarrativeHistory();
```

Per eliminare una parte della cronologia narrativa, passa un numero per rimuovere i primi N elementi:

```typescript
// Delete the first 2 elements
stepHistory.removeNarrativeHistory(2);
```

## Altre funzionalità

<Accordions>

<Accordion title="limit_saved_steps" id="limit-saved-steps">

Ad ogni `passo` vengono salvate tutte le informazioni sullo stato attuale del gioco.

Per evitare che il file di salvataggio diventi troppo grande, è previsto un limite al numero di `steps` salvati. Per impostazione predefinita, vengono salvati solo gli ultimi 20 `steps`, ma è possibile aumentare questo limite (ad esempio a 100). Una volta raggiunto il limite, vengono conservate solo le informazioni essenziali dei `steps` precedenti. Ciò consente di visualizzare l'intera <DynamicLink href="/start/history">cronologia narrativa</DynamicLink>, ma non è possibile <DynamicLink href="/start/labels-flow#go-back">tornare</DynamicLink> a un `passaggio` specifico oltre il limite.

È possibile modificare il limite di salvataggio `step` impostando la proprietà `stepLimitSaved` nell'oggetto `stepHistory`.

```typescript
import { stepHistory } from '@drincs/pixi-vn'

stepHistory.stepLimitSaved = 100
```

Per disattivare il limite di salvataggio `step`, impostare `stepLimitSaved` su `Infinity`.

```typescript
import { stepHistory } from '@drincs/pixi-vn'

stepHistory.stepLimitSaved = Infinity
```
