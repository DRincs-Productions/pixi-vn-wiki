캔버스에 추가되는 모든 컴포넌트에는 별칭 (`alias`)을 지정해야 합니다. 별칭 (`alias`)은 고유한 문자열로 컴포넌트를 참조하는 방법입니다.

<Accordions>

<Accordion title="PixiJS에서 `alias`는 무엇에 해당하나요?" id="what-is-an-alias">

별칭 (`alias`)은 `PixiJSComponent.label`에 해당하므로 `label`을 직접 변경하지 말고, <DynamicLink href="/start/canvas-functions">Pixi’VN이 제공하는 메서드</DynamicLink>를 사용하여 별칭 (`alias`)을 변경하세요.

</Accordion>

</Accordions>

이미 존재하는 별칭 (`alias`)을 지정하여 컴포넌트를 추가하면 새 컴포넌트가 기존 컴포넌트를 대체합니다.

## 상속 인자

기존 별칭 (`alias`)을 사용하여 컴포넌트를 추가하면, 새 컴포넌트는 기존 컴포넌트를 대체하는 것 외에도 기존 컴포넌트의 속성, `zIndex`, 그리고 <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink>를 상속합니다.

## 편집

캔버스 컴포넌트의 별칭 (`alias`)을 편집하려면 [`canvas.editAlias`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#editalias)를 사용할 수 있습니다. 해당 별칭 (`alias`)에 하나 이상의 <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink>가 연결되어 있으면 ticker에서 자동으로 업데이트됩니다.

```ts
import { canvas } from "@drincs/pixi-vn";

canvas.editAlias("sprite1", "sprite2");
```

## 기타 기능

<Accordions>

<Accordion title="게임 레이어 별칭" id="game-layer-alias">

Pixi’VN에서 게임 레이어는 모든 게임 요소가 렌더링되는 메인 게임 영역을 나타내는 특수 컴포넌트입니다.

이 컴포넌트에는 특별한 별칭 (`alias`)인 [`CANVAS_APP_GAME_LAYER_ALIAS`](/jsdoc/pixi-vn/index/variables/CANVAS_APP_GAME_LAYER_ALIAS)가 지정되어 있으며, 스크립트에서 게임 레이어를 참조하는 데 사용됩니다.

이는 전체 레이어에 <DynamicLink href="/start/canvas-animations-effects">애니메이션이나 효과</DynamicLink>를 적용하려는 경우 매우 유용합니다. 삭제와 같이 이 항목에서 허용되지 않는 기능도 있습니다.

```ts
import { CANVAS_APP_GAME_LAYER_ALIAS, shakeEffect } from "@drincs/pixi-vn";

shakeEffect(CANVAS_APP_GAME_LAYER_ALIAS);
```

</Accordion>
