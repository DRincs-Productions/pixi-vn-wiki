title: Почему Pixi'VN?
---

Причина, почему появился Pixi’VN, заключается в том, что текущие системы создания визуальных новелл или 2D RPG основаны на старых системах и имеют множество недостатков.

Pixi’VN включает создание различных 2D-игр в одном движке, приближая создание видеоигр к веб-приложениям, чтобы воспользоваться совместимостью и облачными сервисами.

## Что такое Pixi'VN?

Pixi'VN - универсальный и мощный движок для 2D-игр. Он основан на JavaScript/TypeScript и [PixiJS](https://pixijs.com/).

Он предоставляет следующие возможности:

- ведение повествования
- предоставляет 2D холсты
- предоставляет функционал для воспроизведения звуков и музыки
- хранилище для установки и получения переменных игры.
- сохраняет текущее состояние всей игры на каждом "шаге сюжета", давая возможность вернуться назад
- функционал для сохранения и загрузки текущего состояния игры.

For a quick start, various <DynamicLink href="/start/getting-started#project-initialization">project templates</DynamicLink> are available. Менее опытные разработчики могут использовать эти шаблоны без особых знаний JavaScript/TypeScript.

У вас есть возможность использовать различные типы языков для повествования (в дополнение к JavaScript/TypeScript). В настоящее время вы можете использовать следующее:

## Особенности Pixi'VN

Большая **универсальность** обусловлена тем, что Pixi'VN - это пакет npm, а не фреймворк. This means that it can be installed on any JavaScript project and coupled with your favorite JavaScript framework (React, Vue, etc.).
Вы можете использовать предоставленную функциональность в различных целях, от создания визуальной новеллы или другого типа 2D-игры (например, приключений в стиле point-and-click, RPG и т. д.) используя только повествовательные функции в 3D-игре, демонстрации анимации на веб-сайте и т. д.

Чтобы быть более **легким**, он занимается только конкретными функциями, добавляя или предоставляя возможность добавить больше с помощью других библиотек. Нет необходимости использовать тяжелые IDE для разработки, подойдет любой редактор кода.

Он **не изобретает вещи**, которые уже существуют. Pixi'VN объединяет несколько очень популярных библиотек и предоставляет API для взаимодействия с ними и полного доступа к ним. Он не изобретает новые языки программирования/повествования.
