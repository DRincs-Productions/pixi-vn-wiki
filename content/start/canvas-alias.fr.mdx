Chaque composant ajouté au canvas doit se voir attribuer un `alias`. Un `alias` est un moyen de faire référence à un composant par une chaîne unique.

<Accordions>

<Accordion title="Qu'est-ce qu'un alias dans PixiJS ?" id="what-is-an-alias">

L'`alias` correspond à `PixiJSComponent.label`, donc ne modifiez pas le `label`, mais utilisez les <DynamicLink href="/start/canvas-functions">méthodes fournies par Pixi’VN</DynamicLink> pour modifier l'`alias`.

</Accordion>

</Accordions>

Si un composant est ajouté en lui attribuant un `alias` existant, le nouveau composant remplacera l'ancien.

## Facteur d'hérédité

Si un composant est ajouté en utilisant un `alias` existant, le nouveau composant, en plus de remplacer l'ancien, héritera des propriétés, du `zIndex` et des <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink> de l'ancien composant.

<HeredityFactorExample />

## Modifier

Pour modifier l'`alias` d'un composant du canvas, vous pouvez utiliser [`canvas.editAlias`](/jsdoc/pixi-vn/index/interfaces/CanvasManagerInterface#editalias). Si l'alias possède un ou plusieurs <DynamicLink href="/start/canvas-tickers">tickers</DynamicLink> associés, il sera automatiquement mis à jour dans le ticker.

```ts
import { canvas } from "@drincs/pixi-vn";

canvas.editAlias("sprite1", "sprite2");
```

## Autres fonctionnalités

<Accordions>

<Accordion title="Alias du calque de jeu" id="game-layer-alias">

Dans Pixi’VN, le calque de jeu est un composant spécial qui représente la zone de jeu principale où tous les éléments du jeu sont rendus.

Ce composant s'est vu attribuer un `alias` spécial, [`CANVAS_APP_GAME_LAYER_ALIAS`](/jsdoc/pixi-vn/index/variables/CANVAS_APP_GAME_LAYER_ALIAS), qui est utilisé pour référencer le calque de jeu dans le script.

C'est très utile si vous souhaitez exécuter des <DynamicLink href="/start/canvas-animations-effects">animations ou effets</DynamicLink> sur l'ensemble du calque. Certaines fonctionnalités ne sont pas autorisées sur cet élément, comme la suppression.

```ts
import { CANVAS_APP_GAME_LAYER_ALIAS, shakeEffect } from "@drincs/pixi-vn";

shakeEffect(CANVAS_APP_GAME_LAYER_ALIAS);
```

</Accordion>
