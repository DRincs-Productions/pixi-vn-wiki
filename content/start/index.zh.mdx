您可以通过[初始化新项目](#project-initialization)或在现有项目中[安装软件包](#installation)来开始使用 Pixi’VN。

## 前提条件

开始之前，您必须安装以下工具：

- [Node.js](https://nodejs.org/) 版本 18 或更高。
- 具有 TypeScript 支持的文本编辑器，如：
  - [Visual Studio Code](https://code.visualstudio.com/)
  - [Cursor](https://www.cursor.com/)
  - [VSCodium](https://vscodium.com/)
- (推荐) [Git](https://git-scm.com/)
  - 一个 [GitHub 账户](https://github.com/)

## 项目初始化

如果您想从新项目开始，可以使用以下命令初始化一个使用 Pixi’VN 模板的新项目：

```sh tab="npm"
npm create pixi-vn@latest
```

```sh tab="yarn"
yarn create pixi-vn
```

```sh tab="pnpm"
pnpm create pixi-vn
```

```sh tab="bun"
bun create pixi-vn
```

```sh tab="deno"
deno init --npm pixi-vn
```

您可以在<DynamicLink href="/start/templates">这里</DynamicLink>查看可用模板和互动演示的列表。

项目初始化后，请使用您的文本编辑器（推荐使用 VSCode）打开项目目录，并开始开发您的项目。

所有模板都包含一个 `README.md` 文件，其中有关于项目的更多信息。

## 安装

要在现有的 JavaScript 项目中安装 Pixi’VN 包，请使用以下命令中的一条：

```sh tab="npm"
npm install @drincs/pixi-vn
```

```sh tab="yarn"
yarn add @drincs/pixi-vn
```

```sh tab="pnpm"
pnpm add @drincs/pixi-vn
```

```sh tab="bun"
bun add @drincs/pixi-vn
```

```sh tab="deno"
deno install npm:@drincs/pixi-vn
```

现在您必须在使用引擎之前[初始化 Pixi’VN](#project-initialization)。 例如，您可以将以下代码添加到 `main.ts` 或 `index.ts`（具体取决于您的项目配置）。

### Initialize

在使用 Pixi’VN 引擎之前，您必须初始化游戏。 您可以通过调用 `Game.init` 方法来实现此操作。

This function has the following parameters:

- `element`: 要将画布附加到的 HTML 元素。
- `options`: 相当于您在初始化 [PixiJS 应用程序](https://pixijs.com/8.x/guides/basics#creating-an-application) 时可以使用的选项。 以下选项是必需的：
  - `width`: 画布的宽度。
  - `height`: 画布的高度。
- `devtoolsOptions`: 相当于您在初始化 <DynamicLink href="/start/canvas#use-pixijs-devtools-with-pixivn">PixiJS Devtools</DynamicLink> 时可以使用的选项。

```ts title="src/main.tsx"
import { Game } from "@drincs/pixi-vn";

// Canvas setup with PIXI
const body = document.body
if (!body) {
    throw new Error('body element not found')
}

Game.init(body, {
    height: 1080,
    width: 1920,
    backgroundColor: "#303030",
}).then(() => {
    // ...
});

// read more here: https://pixi-vn.web.app/start/other-narrative-features.html#how-manage-the-end-of-the-game
Game.onEnd(async (props) => {
    Game.clear();
    props.navigate("/");
});

Game.onError((type, error, { notify }) => {
    notify("allert_error_occurred");
});
```
