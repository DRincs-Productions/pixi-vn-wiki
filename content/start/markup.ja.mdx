マークアップ言語とは、ドキュメント内の構造、書式、および関係を定義するテキストエンコードシステムです。 マークアップはドキュメントの表示方法を制御したり、自動処理のためにコンテンツを強化したりするために使用されます。 代表的なマークアップ言語には、HTML、KML/KMZ、XML、Markdown などがあります。

Pixi’VN では、そのシンプルさと普及度から <DynamicLink href="/start/markup-markdown">**Markdown**</DynamicLink> の使用を推奨します。 Markdown は HTML と組み合わせて使用できることが多く、CSS によるカスタムテキストスタイリングが可能です。 高度なスタイリングには、<DynamicLink href="/start/markup-tailwindcss">**Tailwind CSS**</DynamicLink> などの CSS フレームワークの使用を検討してください。
