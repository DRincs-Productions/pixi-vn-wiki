  Estensione non ufficiale di VS Code per la modifica e il testare in tempo reale degli script di ink con Pixi'VN. Fornisce evidenziazione della sintassi, selezione del motore e anteprima in tempo reale per le tue narrazioni.

Se utilizzi *ink*, ti consigliamo di utilizzare l'estensione *ink* da noi sviluppata. Oltre a fornire le funzionalit√† standard del linguaggio *ink*, ti consente di selezionare `pixi-vn` come motore di gioco e di testare la tua narrazione in tempo reale.

![Estensione ink VS Code](https://github.com/user-attachments/assets/cc17384a-7f2f-4e86-b99a-efbf823269d9)

Ecco il file di impostazioni consigliato:

```json title="settings.json"
{
    "ink.mainFile": "start.ink",
    "ink.rootFolder": "src/ink",
    "ink.markup": "Markdown",
    "ink.engine": "pixi-vn"
}
```
