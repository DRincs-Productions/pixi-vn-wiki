---
title: Использование паузы в ink
---

После выполнения сценария # система выполняет команду [go next](/start/labels-flow#next-step) для перехода к следующим шагам. Например, в этом случае вы увидите изображение и текст диалога.

```ink
=== start ===
# image add "image" /image.jpg
Привет, мир!
-> DONE
```

Вы можете использовать `паузу`, чтобы остановить выполнение шагов и установить пустой текст диалога.

The syntax is as follows:

`#` + `пауза`

```ink
=== start ===
# image add "image" /image.jpg
# pause
Привет, мир!
-> DONE
```
