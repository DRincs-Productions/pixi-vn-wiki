The **_ink_ + Pixi’VN integration** provides some additional features to facilitate the use of components that introduce a <DynamicLink href="/start/markup-markdown">Markup</DynamicLink> in the text.

## New Lines

To create a new line, you can use the escape character `\\n`.

<Callout>

In our case, since we use Markdown to display the line break, we will need two `\\n\\n`.

</Callout>

Ad esempio:

```ink title="ink"
=== start ===
Hello, this is a test. \\n\\n This is a new line.
```

## Markdown

On Pixi’VN it is recommended to use Markdown to add style to your text. If you have <DynamicLink href="/start/markup-markdown">implemented Markdown in your project</DynamicLink> then you will be able to use Markdown syntax to style your text in **_ink_**.

To do this you need to keep in mind that many symbols of Markdown syntax are also used by **_ink_ syntax**, such as: `#`, `*`, `/`, `~`, `-`, `|` etc. To avoid conflicts you can use the escape character `\` before the Markdown symbol.

Ad esempio:

```ink title="ink"
Hello, this is some *italic* text and this is some **bold** text.
```

## CSS

You can also use inline CSS to format text using HTML.

Ad esempio:

```ink title="ink"
<span style="color:blue">some *blue* text</span>.
<span style="color:red">some *red* text</span>.
<span style="color:green">some *green* text</span>.
```
