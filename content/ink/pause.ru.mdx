After executing a # script the system does a <DynamicLink href="/start/labels-flow#next-step">go next</DynamicLink> to move to the next steps. Например, в этом случае вы увидите изображение и текст диалога.

```ink
=== start ===
# show image alien eggHead
Hello, world!
-> DONE
```

Вы можете использовать `паузу`, чтобы остановить выполнение шагов и установить пустой текст диалога.

Синтаксис таков:

`#` + `пауза`

```ink tab="ink/start.ink"
=== start ===
# show image alien eggHead
# pause
Hello, world!
-> DONE
```

```ts tab="assets/manifest.ts"
import { AssetsManifest } from "@drincs/pixi-vn";

/**
 * Manifest for the assets used in the game.
 * You can read more about the manifest here: https://pixijs.com/8.x/guides/components/assets#loading-multiple-assets
 */
const manifest: AssetsManifest = {
    bundles: [
        {
            name: "start",
            assets: [
                {
                    alias: "eggHead",
                    src: "https://pixijs.com/assets/eggHead.png",
                },
            ],
        },
    ],
};
export default manifest;
```
