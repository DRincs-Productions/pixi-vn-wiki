---
title: Как перевести текст ink?
---

As explained in more detail <DynamicLink href="/start/translate">here</DynamicLink>, Pixi’VN gives the possibility to translate the game text using a library, such as [i18next](https://www.i18next.com/). Также в **_ink_ + интеграции Pixi'VN** вы можете использовать библиотеку для перевода текста **_ink_**.

Pixi’VN gives the developer the ability to intercept the translation event with the `onInkTranslate` function.

Функция `onInkTranslate` имеет в качестве параметра функцию обратного вызова, которая принимает в качестве параметра текст для перевода и возвращает переведенный текст.

```ts tab="index.ts"
import { onInkTranslate } from '@drincs/pixi-vn-ink'
import { useTranslation } from "react-i18next";

const { t } = useTranslation(["narration"]);
onInkTranslate((text) => {
   return t(text)
})
```

```

import { useI18n } from './i18n'; // [!code focus]

    );
}
```
