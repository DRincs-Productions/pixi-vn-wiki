**Inky** è l'editor ufficiale per gli script _ink_, sviluppato da Inkle (i creatori di _ink_). Scaricalo dal [sito ufficiale](https://www.inklestudios.com/ink/).

![Inky](/inky.png)

## Why?

Writing and testing your narrative in Inky is much faster than running your own web project. Inky ti offre una panoramica chiara di tutti i percorsi ramificati e della logica della storia.

<Callout type="info">

Funzionalità specifiche di Pixi'VN (come sprite dei personaggi, sfondi, musica, ecc.) vengono ignorati in Inky.

</Callout>

Per utilizzare Inky, crea un file denominato `main.ink` nel tuo progetto e includi il file ink che vuoi eseguire. Quindi apri `main.ink` in Inky.

<Callout title="Templates" type="info">

In tutti i modelli è già incluso un file di base `main.ink`. Puoi iniziare subito a scrivere la tua storia.

</Callout>

```ink title="main.ink"
INCLUDE ink/start.ink
-> start
```
