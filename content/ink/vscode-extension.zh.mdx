  一款非官方的VS Code扩展插件，专为编辑与实时测试基于Pixi'VN的ink脚本而设计。 该插件提供了语法高亮、引擎选择及叙事内容的实时预览功能。
---

如果你正在使用*ink*脚本，我们强烈建议你使用我们专门开发的*ink*扩展插件。 除了提供*ink*脚本的语音功能支持外，该插件还允许你将`pixi-vn`选为游戏引擎，并实时预览与测试你的叙事内容。

你可以在[此处](https://marketplace.visualstudio.com/items?itemName=drincs-productions.pixi-vn-ink-vscode)下载插件并获取更多详细信息。

![ink VS Code Extension](https://github.com/user-attachments/assets/cc17384a-7f2f-4e86-b99a-efbf823269d9)

以下是推荐的配置文件：

```json title="settings.json"
{
    "ink.mainFile": "start.ink",
    "ink.rootFolder": "src/ink",
    "ink.markup": "Markdown",
    "ink.engine": "pixi-vn"
}
```
