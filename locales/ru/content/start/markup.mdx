---
title: Язык разметки
---

Язык разметки - это система кодирования текста, определяющая структуру и форматирование документа, а также потенциально возможные связи между его частями. Разметка может управлять отображением документа или обогащать его содержимое для облегчения автоматизированной обработки. Существуют различные типы языков разметки, такие как HTML, KML/KMZ, XML, Markdown и другие.

Из-за простоты и популярности рекомендуется использовать [**Markdown**](/start/markup-markdown.md). Кроме того, поскольку часто возможно комбинировать Markdown с HTML, можно использовать CSS для стилизации текста. Рекомендуется использовать CSS-фреймворк, например [**Tailwind CSS**](/start/markup-tailwindcss).
